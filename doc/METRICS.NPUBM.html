DEPRECATED: Number of "public" methods [METRICS.NPUBM]



DESCRIPTION 


Rules from this category are deprecated - see METRIC category.

The number of methods in a class indicates the amount
of functionality implemented by a class.
NOTES 

Jtest will report an error if the value is greater or equal 20.
REFERENCES 

Mark Schroeder: "A Practical Guide to Object-Oriented Metrics"
IT Pro, November/December, 1999.
