APSC_DV.000460 The application must enforce approved authorizations for logical access to information and system resources in accordance with applicable access control policies. [APSC_DV.000460]

RULES

Avoid defining multiple security roles with the same name in 'web.xml' files [APSC_DV.000460.DSR]
Include an appropriate '<login-config>' element to specify the type of authentication to be performed in 'web.xml' files [APSC_DV.000460.LCA]
Avoid EJB 3 methods without security annotations [APSC_DV.000460.PERMIT]
Ensure that each security role referenced in a 'web.xml' file has a corresponding definition [APSC_DV.000460.SRCD]
