API4-Lack of Resources & Rate Limiting [OWASP2019.API4]

RULES

Close resources as early as possible [OWASP2019.API4.CRWD]
Avoid using the DriverManagerDataSource class in production code [OWASP2019.API4.DMDS]
Do not call the 'start()' method directly on Thread class instances [OWASP2019.API4.ISTART]
Ensure resources are deallocated [OWASP2019.API4.LEAKS]
Specify a valid 'maxlength' value for each form field in Struts validation files [OWASP2019.API4.MLFF]
Ensure 'wait()', 'notify()' and 'notifyAll()' are invoked on an object that is clearly synchronized in its enclosing mod scope [OWASP2019.API4.NSYN]
Do not open or close JDBC connections in loops [OWASP2019.API4.ODBIL]
Ensure sufficient protection against multiple failed authentication attempts [OWASP2019.API4.PBFA]
Validate potentially tainted data before it is used to determine the size of memory allocation [OWASP2019.API4.TDALLOC]
