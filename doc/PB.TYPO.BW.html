Avoid unnecessary self computation [PB.TYPO.BW]



DESCRIPTION 


This rule identifies when variable is self computation, e.g x & x,
x | x, x - x, x / x. An error is reported for each occurrence.
SINCE 

v8.2
BENEFITS 

It may be a typo; otherwise, it is unnecessary to do this self 
computation because the value can be calculated statically.
EXAMPLE 

package examples.rules.pb.typo;

public class BW {

    public int myCalculation(int x) {
        return x - x - 10;  // VIOLATION
    }
}
REPAIR 

Remove 'x - x' because it won't affect the calculation

package examples.rules.pb.typo;

public class BWFixed {
 
    public int myCalculation(int x) { // unused variable x
        return -10; // FIXED
    }
