Avoid method duplication [CDD.DUPM]



DESCRIPTION 


This rule identifies code with duplicate methods. An error is reported for each
occurrence.
SINCE 

v8.0
PARAMETERS 

- Minimal number of duplicated tokens (default value is 80)
- Minimal duplicate tokens size for methods in same hierarchy(default is 15)
- Ignore identifies (default is off)
- Ignore string and character literals (default is on)
- Ignore number literals (default is on)
- Ignore boolean literals (default is on)
- Ignore code in deprecated methods (default is on)
- Ignore methods in deprecated classes (default is on)
BENEFITS 

Copying and pasting code increases the cost of code maintenance because each
time a modification is required, multiple instances of the same code need
to be identified and then modified. Reducing or eliminating the amount of
duplicated code can prevent this problem.
