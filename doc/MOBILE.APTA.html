Avoid using array of primtive wrapper objects [MOBILE.APTA]



DESCRIPTION 


This rule will flag a violation if an array type is primitive wrapper class.
SINCE 

v9.0
BENEFITS 

Use array of primitive type instead of array of primitive wrapper class can 
avoid too many object creation.
EXAMPLE 


package examples.rules.mobile;

public class APTA {

    public int getTotal(Integer[] iarray) {  // VIOLATION
        int total = 0;
        for (int i = 0; i < iarray.length; i++) {
            total += iarray[i].intValue();
        }
        return total;
    }  
}
REPAIR 


package examples.rules.mobile;

public class APTAFixed {

    public int getTotal(int[] iarray) {  // FIXED
        int total = 0;
        for (int i = 0; i < iarray.length; i++) {
            total += iarray[i];
        }
        return total;
    }  
}
