Make the return type "void" for the 'ejbPostCreate()' method [EJB.RTP]



DESCRIPTION 


This rule identifies 'ejbPostCreate()' methods in "EntityBeans" which do not have
"void" for the return type. An error is reported for each occurrence.

METHOD
SINCE 

v4.5
