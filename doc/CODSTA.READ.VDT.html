Do not declare multiple variables of different types in one statement [CODSTA.READ.VDT]



DESCRIPTION 


This rule identifies single statements that declare multiple variables of 
different types. An error is reported if a variable declaration statement 
declares variables that are not of a single type.
SCOPE LEVEL 
LINE
SINCE 

v4.5
BENEFITS 

Declaring multiple variables of different types in a single declaration
statement is confusing and can make code error-prone.
EXAMPLE 

package examples.rules.codsta.read;

public class VDT {

    private int index, index1[]; // VIOLATION

    public void method () {
        int aaa, bbb[];  // VIOLATION
        int ccc;
        int ddd;
    }
}
REPAIR 

Use separate declaration statements for variables of different types.

package examples.rules.codsta.read;

public class VDTFixed {

    private int index;  // FIXED
    private int index1[]; // FIXED

    public void method () {
        int aaa;  // FIXED
        int bbb[];  // FIXED
        int ccc;
        int ddd;
    }
}
