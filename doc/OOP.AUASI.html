Do not implement or extend annotations [OOP.AUASI]



DESCRIPTION 


This rule will flag a violation for any case where an annotation type is 
implemented or extended by a class, interface, or enum type.
SCOPE LEVEL 
LINE
SINCE 

v8.0
BENEFITS 

While it is possible to use an annotation as a super interface for another type,
it is considered bad practice to do so.  Using an annotation as a super 
interface is usually meaningless.  Doing so may indicate a bug or a 
misunderstanding of how annotations are supposed to be used.
EXAMPLE 

package examples.rules.oop;

import java.lang.annotation.Annotation;

public class AUASI implements Deprecated {                //VIOLATION
    public Class<? extends Annotation> annotationType() {
        return Deprecated.class;
    }
}
REPAIR 

Modify the code so that it does not use an annotation type as a super interface.

package examples.rules.oop;

@Deprecated
public class AUASI_2 {  //FIXED
}
