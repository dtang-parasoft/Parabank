Do not throw 'java.rmi.RemoteException' in a bean's local interface and local home interface [EJB.RILH]



DESCRIPTION 


This rule identifies methods of session beans' and entity beans' local interface 
and local home interface that throw "java.rmi.RemoteException".  An error is 
reported for each occurrence.
SCOPE LEVEL 

METHOD
SINCE 

v4.5
EXAMPLE 

package examples.rules.ejb;

import java.rmi.RemoteException;
import javax.ejb.EJBLocalHome;

public interface RILH extends EJBLocalHome {
    public String create(String s) throws RemoteException;  //VIOLATION
}
REPAIR 

Do not declare the methods in the local home interface to throw "java.rmi.
RemoteException".

package examples.rules.ejb;

import java.rmi.RemoteException;
import javax.ejb.EJBLocalHome;

public interface RILH extends EJBLocalHome {
    public String create(String s);  //FIXED
}
REFERENCES 

Sections 7.10.7 (p. 98), 7.10.8 (p. 98), and 9.10 (p. 122) of the EJB 
specification (version 2.0).  The EJB specification is available online at 
http://download.oracle.com/otndocs/jcp/7294-ejb-2.0-fr2-spec-oth-JSpec/
