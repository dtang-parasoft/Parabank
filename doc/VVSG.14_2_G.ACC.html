Remove commented out Java code [VVSG.14_2_G.ACC]



DESCRIPTION 


This rule identifies commented java code. An error is reported for each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v8.2
PARAMETERS 

-Ignore comments prefaced by "TODO" (default on)
BENEFITS 

Removing the commented java code will reduce the class size and also increase 
the readability of the class.
EXAMPLE 

package examples.rules.uc;

public class ACC {

// VIOLATION 
//	public static void addElement(List list, Object value) {
//		if (!list.contains(value)) {
//			list.add(value);
//		}
//	}
    
    public static void insertElement(List list, Object value) {
        if (!list.contains(value)) {
            list.add(0, value);
        }
    }
}
REPAIR 

Removing the commented java code.

package examples.rules.uc;

public class ACCFixed {
    
    // FIXED : remove the commented java code
    
    public static void insertElement(List list, Object value) {
        if (!list.contains(value)) {
            list.add(0, value);
        }
    }
}
