APSC_DV.000500 The application must prevent non-privileged users from executing privileged functions to include disabling, circumventing, or altering implemented security safeguards/countermeasures. [APSC_DV.000500]

RULES

Avoid using anonymous "privileged" classes when invoking "AccessController.doPrivileged()" [APSC_DV.000500.ACDP]
Declare subclasses of 'PrivilegedAction', 'PrivilegedExceptionAction', and 'PrivilegedActionException' "final" [APSC_DV.000500.PAF]
Wrap "privileged" method invocations in "final" methods [APSC_DV.000500.PCFM]
Limit the number of lines in "privileged" code blocks [APSC_DV.000500.PCL]
Wrap "privileged" method invocations in "private" methods [APSC_DV.000500.PCPM]
Avoid using dynamically loaded classes in "privileged" code blocks [APSC_DV.000500.PDLC]
Ensure that all Permissions, PrivilegedActions, and PrivilegedActionExceptions are declared in the same package [APSC_DV.000500.PPKG]
Avoid operating on tainted data in privileged blocks [APSC_DV.000500.PRIVIL]
Declare the 'run()' method of 'PrivilegedAction' and 'PrivilegedExceptionAction' implementations "final" [APSC_DV.000500.PRMF]
