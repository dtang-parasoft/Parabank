Do not leave spaces between qualified names and method invocations [APSC_DV.003215.DOT]



DESCRIPTION 


This rule identifies qualified names and method invocations that have whitespaces
in between. An error would be reported for each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v8.0
BENEFITS 

It is coding convention in general to have no space separates names or
method invocations. Extra space might reduce the readability of the code
and confuse other programmers.
EXAMPLE 

package examples.rules.format;

public class DOT {
    java. util. Vector v; //VIOLATION
    private void printALine() {
        System . out . println("A line"); //VIOLATION
    }
}
REPAIR 

Remove the whitespaces

package examples.rules.format;

public class DOTFixed {
    java.util.Vector v; //FIXED
    private void printALine() {
        System.out.println("A line"); //FIXED
    }
}
