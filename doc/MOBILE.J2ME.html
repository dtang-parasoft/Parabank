Java 2 Micro Edition [MOBILE.J2ME]

RULES

Do not use anonymous classes as interface implementors [MOBILE.J2ME.ACII]
Do not use an array length in a loop condition expression [MOBILE.J2ME.ARLL]
Avoid constant initializations of primitive arrays that exceed a certain size [MOBILE.J2ME.CIPA]
Avoid classes that are subclassed only once and are not publicly used [MOBILE.J2ME.CSOO]
Do not access a field excessively [MOBILE.J2ME.EAOF]
Ensure methods use return parameters instead of returning new objects [MOBILE.J2ME.EURP]
Catch 'OutOfMemoryError' for large array allocations [MOBILE.J2ME.OOME]
