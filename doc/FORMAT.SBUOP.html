Enforce number of space character(s) between a postfixed unary operator and its operand [FORMAT.SBUOP]



DESCRIPTION 

    
This rule identifies code that contains white space between a unary operator 
and its operand. It applies to postfix "++" and "--" operators. An error is 
reported for each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v4.5
PARAMETERS 

-Number of spaces before unary operators (default value is 0)
BENEFITS 

Space between a unary operator and its operand can make code confusing and error-prone.
EXAMPLE 

package examples.rules.format;

public class SBUOP {
    public void method (int x) {
        System.out.println("x: " + x ++);  // VIOLATION
    }
   
}
REPAIR 

Remove the space.

package examples.rules.format;

public class SBUOPFixed {
    public void method (int x) {
        System.out.println("x: " + x++);  // FIXED
    }
   
}
