Ensure that the type that has the same name as the file is declared "public" [PB.PCSF]



DESCRIPTION 


The rule identifies any class or interface that has the same
name as the file, but is not declared "public".
An error is reported for each occurrence.
SCOPE LEVEL 

CLASS
SINCE 

v6.0
NOTES 

Errors will not be reported for inner classes.
EXAMPLE 

file name: PCSF.java

package examples.rules.pb;

class PCSF //VIOLATION
{}
REPAIR 

file name: PCSFFixed.java

package examples.rules.pb;

public class PCSFFixed  //FIXED
{}
