Use underscore characters (_) in numerical literal [CODSTA.READ.ULIT]



DESCRIPTION 

ï»¿DESCRIPTION
In Java SE 7 and later, underscores (_) may be used in numerical literals to separate groups of digits.
This rule identifies cases where the underscore convention isn't used and reports errors.
SINCE 
v9.4
PARAMETERS 
-Minimum value
    Specifies the minimum value that is checked. Literals with values lower that the minimum are ignored. 
    Default is 100000.
    
-Density
   How often underscores are used. Example: 
   If you set this parameter to 4 then the underscore convention looks like 100_0000_0000
   Default is 3.

-Check only initialized variables
   Limit only to cases where numeric literal is initialized. Example:
   If this parameter is used, literals that look like a = 1000 + 1000000 are ignored.
   The parameter is not used by default.
BENEFITS 
Using an underscore in numerical literals increases code readability.

NOTE
This rule has one quickfix - "Place underscore (_) at correct position in numbers."
EXAMPLE 
int a = 100000000; // VIOLATION
REPAIR 
int a = 100_000_000; // FIXED
