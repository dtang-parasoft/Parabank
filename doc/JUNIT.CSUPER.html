Call the superclass' 'setUp' and 'tearDown' methods in the 'setUp' and 'tearDown' methods of JUnit test classes [JUNIT.CSUPER]



DESCRIPTION 


This rule identifies JUnit test classes that do not call the 
superclass' setup method in the setup method or the superclass' 
tear-down method in the tear-down method. An error is reported for 
each occurrence.
SCOPE LEVEL 

METHOD
SINCE 

v5.1
NOTES 

This rule would not flag classes that directly extend 
'junit.framework.TestCase', as 'setUp' and 'tearDown' methods
in that class do not do anything.

This rule defines a setup method as any method either called 
'setUp()' or (for JUnit versions 4.0 and later) any method with 
the "org.junit.Before" annotation.  Similarly, a tear-down method
is defined as any method either called 'tearDown()' or (for JUnit 
versions 4.0 and later) any method with the "org.junit.After" 
annotation.
BENEFITS 

The setup method must call the superclass' setup method to 
ensure that all of the parent test classes' setup methods  are
performed as well.

The tear-down method must call the superclass' tear-down method to
ensure that all of the parent test classes' tear-down methods  are
performed as well.
EXAMPLE 

package examples.rules.junit;

import junit.framework.TestCase;

public class CSUPER extends TestCase {
    Object testObj;
    
    public void setUp() throws Exception { 
        //NO VIOLATION: this class directly extends TestCase
        testObj= new Object();
    }
    
    public void tearDown() throws Exception {
        //NO VIOLATION: this class directly extends TestCase
        testObj= null;
    }
}

class CSuperSubTest extends CSUPER {
    Object subTestObject;
    
    public void setUp() throws Exception {
        //VIOLATION, does not call super.setUp() 
        subTestObject= new Object();
    }
    
    public void tearDown() throws Exception {
        //VIOLATION, does not call super.tearDown()
        subTestObject= null;
    }
}
REPAIR 

Call super.setUp() and super.tearDown().

package examples.rules.junit;

import junit.framework.*;

public class CSUPER_Fixed extends TestCase {
    Object testObj;
    
    public void setUp() throws Exception { 
        //NO VIOLATION: this class directly extends TestCase
        testObj= new Object();
    }
    
    public void tearDown() throws Exception {
        //NO VIOLATION: this class directly extends TestCase
        testObj= null;
    }
}

class CSuperSubTest_Fixed extends CSUPER_Fixed {
    Object subTestObject;
    
    public void setUp() throws Exception {
        super.setUp();  //FIXED
        subTestObject= new Object();
    }
    
    public void tearDown() throws Exception {
        try { 
            subTestObject= null;
        } finally { 
            super.tearDown();  //FIXED
        } 
    }
}
