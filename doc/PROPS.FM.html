Follow the limit for number of property entries [PROPS.FM]



DESCRIPTION 


This rule identifies any property file that does not satisfy the user-defined
requirement for number of property entries. By default, an error is reported if
property files do not have between 1 and 500 property entries.
SCOPE LEVEL 
LINE
SINCE 

v8.1
PARAMETERS 

-Ignore property file name (regular expression)
    -Allows you to provide a list of property files to ignore. If a property file 
     matches any of the regular expressions, it will be exempt from this rule.
-Upper threshold (default as 500)
-Lower threshold (default as 1)

An error will be reported if the number of property entries is 
beyond the specified thresholds.
