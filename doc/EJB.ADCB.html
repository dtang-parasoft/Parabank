Do not use EJB code in java swing or servlet classes [EJB.ADCB]



DESCRIPTION 


This rule identifies EJB code inside java swing or servlet classes.
An error is reported for each occurrence.
SCOPE LEVEL 

CLASS
SINCE 

v4.5
BENEFITS 

If an entity bean is called remotely from java swing or a servlet,
it will cause a number of remote network calls that increase network
overhead.  Also, session beans are used to implement facades 
for entity beans; exposing the session bean directly to
java swing or servlet creates a tight coupling between java
swing/servlet code and the session bean.
REFERENCES 

"Core J2EE Patterns - Best Practices and Design Strategies"
by Deepak Alur, John Cupri and Dan Malks. pp. 94
