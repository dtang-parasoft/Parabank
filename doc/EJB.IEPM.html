Implement one or more 'ejbPostCreate()' methods in EntityBean classes [EJB.IEPM]



DESCRIPTION 


This rule identifies EntityBean classes that do not implement one or
more 'ejbPostCreate ()' methods. An error is reported for each 
occurrence.
SCOPE LEVEL 

CLASS
SINCE 

v4.5
BENEFITS 
This is required in Enterprise JavaBeans specification version 1.1.
