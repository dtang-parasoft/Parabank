Use spaces instead of tabs (or tabs instead of spaces) [APSC_DV.003215.DUT]



DESCRIPTION 


Depending on the setting of the parameters, this rule identifies code that uses
either tab characters instead of spaces or space characters instead of tabs. By
default, the rule checks that space characters are used instead of tabs.  An
error is reported for each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v4.5
PARAMETERS 

-Choose to use tabs or spaces
    -Depending on the setting of this parameter, the rule will enforce the
     usage of either tabs or spaces.  (Default is "Use spaces".)
    -If the parameter is set to "Use tabs", the rule will only check for the
     use of spaces at the beginning of each line.  A violation will be flagged
     for each block of spaces appearing at the beginning of a line that is
     equivalent to one tab.
-Spaces per tab
    -The number of spaces that are equivalent to one tab (Default is 4.)
BENEFITS 

By avoiding tab characters, you reduce the chances of code appearing
differently in different editors.
EXAMPLE 

package examples.rules.format;

public class DUT {
    public void spaced_method () {}
    public void tabbed_method () {}  // VIOLATION
}
REPAIR 

Replace tabs with an agreed-upon number of spaces.  Traditionally a tab
represents '8' spaces.  The preceding example could occur when a user
has set their editor to use '4' space tabs, causing confusion for people
who view the file in a different editor.

package examples.rules.format;

public class DUTFixed {
    public void spaced_method () {}
    public void nolonger_tabbed_method () {}  // FIXED
}
