DEPRECATED: Number of statements in a method [METRICS.NSTMT]



DESCRIPTION 


Rules from this category are deprecated - see METRIC category.

Counts the number of statements per method.
NOTES 

Jtest will report an error if the value is greater or equal 50.
BENEFITS 

Extremely long methods can be difficult to comprehend and often contain several
conceptually unrelated functions.
REPAIR 

Break the method into several methods so that each accomplishs a single distinct
function.
