CWE-200: Exposure of Sensitive Information to an Unauthorized Actor [CWE.200]

RULES

Do not call the 'printStackTrace()' method of "Throwable" objects [CWE.200.ACPST]
Do not log confidential or sensitive information [CWE.200.CONSEN]
Avoid debug information from Spring Security framework to logs [CWE.200.EWSSEC]
Do not pass exception messages into output in order to prevent the application from leaking sensitive information [CWE.200.PEO]
Prevent exposure of sensitive data [CWE.200.SENS]
Avoid passing sensitive data to functions that write to log files [CWE.200.SENSLOG]
Avoid calling print methods of 'System.err' or 'System.out' [CWE.200.SIO]
