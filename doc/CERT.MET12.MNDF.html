Do not define 'finalize()' method in bean classes [CERT.MET12.MNDF]



DESCRIPTION 



This rule identifies bean classes that define the 'finalize ()' method.
An error is reported for each occurrence.
SCOPE LEVEL 

METHOD
SINCE 

v4.5
