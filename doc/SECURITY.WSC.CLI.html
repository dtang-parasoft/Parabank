Inspect instantiations of 'ClassLoader' objects [SECURITY.WSC.CLI]



DESCRIPTION 


This rule identifies instantiations of 'ClassLoader' object. 
A message is reported for each occurrence.
SINCE 

v6.0
SECURITY RELEVANCE 

category: Weak Security Controls
category: Malicious Code

This rule prevents insecure class loaders. See BENEFITS for more
information.
BENEFITS 

As insecure 'ClassLoader' object could allow untrusted 
classes to be loaded. Consequently, instantiations of 'ClassLoader' 
should be inspected carefully.
EXAMPLE 

package examples.rules.security.wsc;

public class CLI {
    public static void main(String[] args) {
        ClassLoader cl = new MyClassLoader();
    }
}
REPAIR 

Ensure that the class loader is secure.
