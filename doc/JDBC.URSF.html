Use instance of RowSetFactory to create a RowSet object [JDBC.URSF]



DESCRIPTION 

This rule identifies cases where a RowSet object is not created using the RowSetFactory class.
SINCE 
v9.4
BENEFITS 
Java SE 7 and later have RowSet 1.1 with a new layer for JDBC RowSet implementations. 
Using the javax.sql.rowset factory eliminates code dependency on the underlying JRE implementation 
and allows you finetune the implementation when necessary.
 
NOTE
A RowSetFactory implementation must be provided; do not use this rule 
if the JDBC driver does not include a RowSetFactory implementation.
EXAMPLE 
JdbcRowSet rowSet = new JdbcRowSetImpl(); // VIOLATION
REPAIR 
RowSetFactory rowSetFactory = RowSetProvider.newFactory();
JdbcRowSet rowSet = rowSetFactory.createJdbcRowSet(); // FIXED
