A8-Software and Data Integrity Failures [OWASP2021.A8]

RULES

Avoid parsing untrusted data with XMLDecoder [OWASP2021.A8.AUXD]
Create a 'serialVersionUID' for all 'Serializable' classes [OWASP2021.A8.DUID]
Assign 'protected' accessibility to 'readResolve()' and 'writeReplace()' methods in serializable classes [OWASP2021.A8.MASP]
Implement 'readObject()' and 'writeObject()' for all 'Serializable' classes [OWASP2021.A8.OROM]
Always specify absolute paths to execute commands [OWASP2021.A8.PBRTE]
Define a "readResolve" method for all instances of Serializable types [OWASP2021.A8.RRSC]
Ensure that all fields are assigned by the 'readObject()' method and written out by the 'writeObject()' method [OWASP2021.A8.RWAF]
Disable LDAP deserialization [OWASP2021.A8.SC]
Always declare writeObject and readObject methods for Serializable subclasses of non-Serializable parents [OWASP2021.A8.SCBNP]
Do not declare the 'readObject()' method as "synchronized" [OWASP2021.A8.SROS]
Always verify JarFile signatures [OWASP2021.A8.VJFS]
Validate objects before deserialization [OWASP2021.A8.VOBD]
