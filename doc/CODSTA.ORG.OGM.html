Organize methods by name [CODSTA.ORG.OGM]



DESCRIPTION 


This rule identifies methods that are not organized by name. An error is 
reported if methods of the same name are not grouped together.
SCOPE LEVEL 
METHOD
SINCE 

v4.5
BENEFITS 

Makes code easier to read and less error-prone.
EXAMPLE 

package examples.rules.codsta.org;

public class OGM {
    void foo () {}
    void bar () {}
    void foo (int a) {  // VIOLATION: this should be with the previous 'foo ()'
    } 
}
REPAIR 

Move the 'foo' methods together.

package examples.rules.codsta.org;

public class OGMFixed {
    void foo () {}
    void foo (int a) {  // FIXED
    } 
    void bar () {}
}
