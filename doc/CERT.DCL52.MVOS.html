Do not declare multiple variables in one statement [CERT.DCL52.MVOS]



DESCRIPTION 


This rule identifies single statements that declare multiple variables.
An error is reported if one declaration statement declares more than one
variable.
SCOPE LEVEL 
LINE
SINCE 

v4.5
BENEFITS 

Makes code easier to read and less error-prone.
EXAMPLE 

package examples.rules.codsta.read;

public class MVOS {
    String s1, s2; // VIOLATION;

    public MVOS() {
        s1 = "hello";
        s2 = "world";
    }

    public boolean method() {
        int i = 0, j = 0;  // VIOLATION
        return (i == j);
    }

    public String toString() {
        return s1 + " " + s2;
    }
}
REPAIR 

Give each variable its own declaration statement.

package examples.rules.codsta.read;

public class MVOSFixed {
 
    String s1;  // FIXED
    String s2;

    public MVOSFixed() {
        s1 = "hello";
        s2 = "world";
    }

    public boolean method() {
        int i = 0;  // FIXED
        int j = 0;
        return (i == j);
    }

    public String toString() {
        return s1 + " " + s2;
    }
}
