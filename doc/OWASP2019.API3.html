API3-Excessive Data Exposure [OWASP2019.API3]

RULES

Ensure that 'axis.development.system' is set to "false" in Axis 'server-config.wsdd' files [OWASP2019.API3.ADS]
Ensure that 'axis.enableListQuery' is set to "false" in Axis 'server-config.wsdd' files [OWASP2019.API3.AELQ]
Catch all exceptions which may be thrown within Servlet methods [OWASP2019.API3.CETS]
Declare package-private methods as inaccessible as possible [OWASP2019.API3.DPAM]
Ensure that file target paths retrieved by resolving symbolic links are safe [OWASP2019.API3.FOLLOW]
Use 'https' instead of 'http' for the 'transportReceiver' and 'transportSender' in 'axis2.xml' configuration files [OWASP2019.API3.HTTPS]
Do not use static variables in servlets without synchronization [OWASP2019.API3.NSSS]
Do not throw 'NullPointerException' [OWASP2019.API3.NTNPE]
Declare subclasses of 'PrivilegedAction', 'PrivilegedExceptionAction', and 'PrivilegedActionException' "final" [OWASP2019.API3.PAF]
Declare subclasses of 'Permission' and 'BasicPermission' "final" [OWASP2019.API3.PBPSF]
Always specify absolute paths to execute commands [OWASP2019.API3.PBRTE]
Avoid using dynamically loaded classes in "privileged" code blocks [OWASP2019.API3.PDLC]
Do not pass exception messages into output in order to prevent the application from leaking sensitive information [OWASP2019.API3.PEO]
Declare the 'run()' method of 'PrivilegedAction' and 'PrivilegedExceptionAction' implementations "final" [OWASP2019.API3.PRMF]
Enforce 'SecurityManager' checks before setting or getting fields [OWASP2019.API3.SCF]
Enforce 'SecurityManager' checks in methods of 'Cloneable' classes [OWASP2019.API3.SCLONE]
Enforce 'SecurityManager' checks in methods of 'Serializable' classes [OWASP2019.API3.SCSER]
Ensure 'SecurityManager' check in constructor of "public" non-"final" sensitive type [OWASP2019.API3.SCSM]
Store sensitive data in mutable objects [OWASP2019.API3.SDM]
Prevent exposure of sensitive data [OWASP2019.API3.SENS]
Ensure that JavaBean classes implement 'java.io.Serializable' [OWASP2019.API3.SERIALIZABLE]
Avoid using the SOAP Monitor module [OWASP2019.API3.SMM]
Ensure 'SecurityManager' checks before 'Socket' transfers or retrievals [OWASP2019.API3.SMSTR]
Minimize usage of System.out and System.err in Servlets [OWASP2019.API3.SOP]
Declare a package-private method "final" if it is not overridden [OWASP2019.API3.SPAM]
Declare 'serialPersistentFields' "private static final" [OWASP2019.API3.SPF]
Do not interrogate or modify security policy information in a web component [OWASP2019.API3.SPI]
Declare "transient" fields "private" [OWASP2019.API3.TFP]
Avoid "transient" fields in serialPersistentFields array [OWASP2019.API3.TSPF]
Use the ''getSecure()'' and ''setSecure()'' methods to enforce the use of secure cookies [OWASP2019.API3.UOSC]
Use "PreparedStatements" correctly [OWASP2019.API3.UPSC]
Encapsulate all redirect and forward URLs with a validation function [OWASP2019.API3.VRD]
Properly validate server identity [OWASP2019.API3.VSI]
