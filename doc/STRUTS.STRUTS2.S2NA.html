Ensure each validation file has a corresponding Action [STRUTS.STRUTS2.S2NA]



DESCRIPTION 


This rule identifies if a validation file does not have a corresponding action. 
An error is reported for each occurrence.
SINCE 

v8.4
SECURITY RELEVANCE 

It is easy for developers to forget to update the validation file when an action 
is renamed. If this happens, the out-of-date validation file will become 
useless. The input will then be vulnerable to attackers.
