Do not use "java.awt.peer.*" interfaces directly [PORT.PEER]



DESCRIPTION 


This rule identifies code that directly uses the interfaces defined in 
the "java.awt.peer" package. An error is reported for each occurrence.
SCOPE LEVEL 

FILE
SINCE 

v4.5
BENEFITS 

These interfaces are meant for use only by implementors of the AWT package.
Applications that directly use these interfaces are not portable.
EXAMPLE 

package examples.rules.port;

import java.awt.peer.ComponentPeer;

public interface PEER extends ComponentPeer {    // VIOLATION
    public void setName(String name);
}
REPAIR 

Do not use the 'java.awt.peer' package directly.
REFERENCES 

David Flanagan: "Java in a Nutshell".  O'Reilly
November, 1999: Third Edition, pp.190-192
