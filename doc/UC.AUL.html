Avoid unused labels [UC.AUL]



DESCRIPTION 


This rule identifies unused labels. An error is reported for 
each occurrence.
SINCE 

v6.0
BENEFITS 

Unused labels should be removed because they are unnecessary 
extra code.
EXAMPLE 

package examples.rules.uc;

public class AUL {
    public void myMethod() {
        a: //VIOLATION
    }
}
REPAIR 

package examples.rules.uc;

public class AULFixed {
    public void myMethod() {
	//FIXED
    }
}
