Do not access or set System properties [CWE.20.SYSP]



DESCRIPTION 


This rule identifies code that attempts to access or set 
System properties through "java.lang.System". An error is 
reported for each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v6.0
SECURITY RELEVANCE 

category: Backdoor Vulnerabilities
category: Malicious Code

Code which accesses or sets System properties may be malicious. See BENEFITS for
more information.
BENEFITS 

Accessing and setting System properties is not a trivial 
operation. Usage of System properties should be restricted 
because information and access to the System could be used 
by a malicious programmer.
EXAMPLE 

package examples.rules.security.bv;

public class SYSP {
    public void setClassPath(String clpath){
        System.setProperty("CLASSPATH", clpath); //VIOLATION
        reloadLibraries();
    }
}
REPAIR 

Do not access or set System properties.
