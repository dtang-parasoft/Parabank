A7-Identification and Authentication Failures [OWASP2021.A7]

RULES

Avoid using the 'getRequestedSessionId' method from the 'HttpServletRequest' class [OWASP2021.A7.HGRSI]
Do not rely on IP addresses obtained from HTTP request headers for authentication [OWASP2021.A7.HTTPRHA]
Always call 'HttpSession.invalidate()' before 'LoginContext.login()' [OWASP2021.A7.ISL]
Ensure proper session expiration [OWASP2021.A7.RUIM]
Ensure that sessions are configured to time out in 'web.xml' files [OWASP2021.A7.STTL]
Protect against using unprotected credentials [OWASP2021.A7.TDPASSWD]
