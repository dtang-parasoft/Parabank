OBJ51-J: Minimize the accessibility of classes and their members [CERT.OBJ51]

RULES

Declare "package-private" types as inaccessible as possible [CERT.OBJ51.DPAC]
Declare package-private fields as inaccessible as possible [CERT.OBJ51.DPAF]
Declare package-private methods as inaccessible as possible [CERT.OBJ51.DPAM]
Declare "public/protected" types as inaccessible as possible [CERT.OBJ51.DPPC]
Declare "public/protected" fields as inaccessible as possible [CERT.OBJ51.DPPF]
Declare "public/protected" methods as inaccessible as possible [CERT.OBJ51.DPPM]
