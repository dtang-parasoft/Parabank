Avoid using floats [MOBILE.FLOATER]



DESCRIPTION 


This rule reports violations when a float or double is used in a mathematical operation such as addition, subtraction, division, multiplication, modulus, or has its sign changed.
SCOPE LEVEL 
LINE
SINCE 

v9.0
NOTES 

Some mobile platforms do not have a dedicated fpu, and do all float operations in software, which can be very expensive.
BENEFITS 

This rule notifies the developer that they are doing a potentially expensive calculation.
EXAMPLE 

package examples.rules.mobile;

public class FLOATER {
	float a=4;
	float b=4;
	float violation=a*b;  // VIOLATION
	float violatn=a+b;  // VIOLATION
	float violtn=a-b;  // VIOLATION
	float vltn=a/b;  // VIOLATION
}
