Avoid unnecessary nested blocks [CODSTA.BP.BLOCK]



DESCRIPTION 


This rule identifies nested blocks. An error is reported
for each occurrence.
SINCE 

v6.0
BENEFITS 

Nested blocks are most likely left-over code from debugging.
Since nested blocks could be confusing, they should be
eliminated whenever possible to create more cleanly structured code.
EXAMPLE 

package examples.rules.codsta.bp;

public class BLOCK {
    public void myMethod() {
        { //VIOLATION
            System.out.print("Hello ");
        }
        System.out.println("World!");
    }
}
REPAIR 

package examples.rules.codsta.bp;

public class BLOCKFixed {
    public void myMethod() {
        System.out.print("Hello "); //FIXED
        System.out.println("World!");
    }
}
