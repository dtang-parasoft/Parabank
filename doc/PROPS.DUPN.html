Avoid duplicated property names [PROPS.DUPN]



DESCRIPTION 


This rule identifies any property name that appears more than one in the same property file. An error
is reported for each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v8.1
PARAMETERS 

-Ignore property file name (regular expression)
    -Allows you to provide a list of property files to ignore. If a property file 
     matches any of the regular expressions, it will be exempt from this rule.
BENEFITS 

It is a possible bug if a property name appears more than one in the same property file.
EXAMPLE 

a=A
a=B  // VIOLATION
a=C  // VIOLATION
A=A
REPAIR 

a=A
b=B  // FIXED
c=C  // FIXED
A=A
