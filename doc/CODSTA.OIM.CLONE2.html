Use the 'clone()' method only to implement 'Cloneable' interface [CODSTA.OIM.CLONE2]



DESCRIPTION 


This rule identifies any 'clone()' method that is declared in a class
that does not implement 'Cloneable'. An error is reported for each occurrence.
SCOPE LEVEL 

METHOD
SINCE 

V6.0
BENEFITS 

If a class contains a 'clone()' method which returns an 
object, the class is preferred to be declared as implementing
'Cloneable' since it provides a better description of the 
class's functionalities.
EXAMPLE 

package examples.rules.codsta.oim;

public class CLONE2 {
    public Object clone() { //VIOLATION
        //...
    }
}
REPAIR 

package examples.rules.codsta.oim;

public class CLONE2Fixed implements Cloneable {
    public Object clone() { //FIXED
        //...
    }
}
