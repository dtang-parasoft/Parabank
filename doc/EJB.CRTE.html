Declare 'ejbCreate()' methods "public", but neither "static" nor "final" [EJB.CRTE]



DESCRIPTION 


This rule identifies 'ejbCreate()' methods that are either "final",
"static" or non-"public". An error is reported for each occurrence.
SCOPE LEVEL 

METHOD
SINCE 

v4.5
BENEFITS 

An enterprise bean must have one or more ejbCreate methods.
