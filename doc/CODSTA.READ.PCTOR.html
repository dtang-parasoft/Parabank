Do not declare "public" constructors in non-public classes [CODSTA.READ.PCTOR]



DESCRIPTION 


This rule identifies non-"public" classes with "public" constructors.
An error is reported for each occurrence.
SCOPE LEVEL 

CLASS
SINCE 

v4.5
NOTES 

This rule will ignore all "inner" and "local" classes.
BENEFITS 

Constructors for non-"public" classes are not publicly
accessible.  To avoid confusion, ensure that these constructors 
do not have the "public" modifier.
EXAMPLE 

package examples.rules.codsta.read;

public class PCTOR {   

    class PCTORInner { // package accessible class

        public PCTORInner () {  // VIOLATION
        }
    }
}
REPAIR 

Remove the "public" modifier on any constructors or add it to the class.

package examples.rules.codsta.read;

public class PCTORFixed {  
 
    class PCTORFixedInner { // package accessible class

        PCTORFixedInner () {  // FIXED

        }
    }
}
