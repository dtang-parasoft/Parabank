Avoid code duplication [CDD.DUPC]



DESCRIPTION 


This rule identifies duplicated code. An error is reported for each occurrence.
SINCE 

v8.0
PARAMETERS 

- Minimal number of duplicated tokens (default value is 80)
- Ignore identifiers (default is off)
- Ignore string and character literals (default is on)
- Ignore number literals (default is on)
- Ignore boolean literals (default is on)
- Ignore code in deprecated methods (default is on)
BENEFITS 

Copying and pasting code will increase the cost of code maintenance because each time a modification
is required, multiple instance of the same code will need to be identified and then modified. Reducing or eliminating the amount
of duplicated code can prevent this problem.
