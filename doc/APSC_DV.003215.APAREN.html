Use '()' to separate complex expressions [APSC_DV.003215.APAREN]



DESCRIPTION 


This rule identifies code that does not use '() to separate complex
expressions. An error is reported for each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v5.0
BENEFITS 

Using '()' to separate expressions can help to avoid operator precedence
problems.
EXAMPLE 

package examples.rules.format;

public class APAREN {

    public int max(int i, int j) {
        int default_num = 0;

    if (i >= j && i >= 0) { // VIOLATION
        return i;
    } else if (j > i && j >= 0) { // VIOLATION
        return j;
    } else {
        return default_num;
    }
    }
}
REPAIR 

Use the '()' to separate the expression.

package examples.rules.format;

public class APARENFixed {

    public int max(int i, int j) {
        int default_num = 0;

        if ((i >= j) && (i >= 0)) { // OK 
            return i;
        } else if ((j > i) && (j >= 0)) { // OK 
            return j;
        } else {
            return default_num;
        }
    }
}
