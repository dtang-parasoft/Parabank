Method Calls in Methods [METRIC.NOMCIM]



DESCRIPTION 


This metric calculates the number of method calls within a method. 
A high number of invocations may indicate that the design is too complex
and can be simplified by splitting the method into smaller methods.
SINCE 

v10.4.3
NOTES 

Method calls defined in lambdas and other code entities within a method body
are included in the total number of invocations within this method.
BENEFITS 

This metric helps you reduce design complexity and improve code readability.
