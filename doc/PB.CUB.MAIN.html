Use the method name 'main()' only for the entry point method [PB.CUB.MAIN]



DESCRIPTION 


This rule identifies methods named 'main()' that do not have the signature
'public static void main (java.lang.String[])'. An error is reported for 
each occurrence.
SCOPE LEVEL 

METHOD
SINCE 

v4.5
BENEFITS 

'main()' is the name for a method with a special meaning in Java. To prevent
confusion, do not use 'main()' unless you are defining
'public static void main (java.lang.String[])'.
EXAMPLE 

package examples.rules.pb.cub;

public class MAIN {

    public static void main (String[] args) {
        System.out.println ("This is the main method.");
    }

    public static void main () {  // VIOLATION
        System.out.println ("This is another method named main.");
    }
}
REPAIR 

Rename the method to something other than 'main()'.

package examples.rules.pb.cub;

public class MAINFixed {

    public static void main (String[] args) {
        System.out.println ("This is the main method.");
    }

    public static void other_main () {  // FIXED
        System.out.println ("This is another method, now named other_main.");
    }
}
