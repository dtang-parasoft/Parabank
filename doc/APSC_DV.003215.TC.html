Avoid using trailing comments [APSC_DV.003215.TC]



DESCRIPTION 



This rule identifies any usage of trailing comments. An
error is reported for each occurrence.
SINCE 

v6.0
BENEFITS 

Trailing comments are generally hard to maintain. They
must be manually aligned whenever the code changes. They
also extend the lengths of existing lines of codes, and 
such an extension might lead to an oversized lines.
EXAMPLE 

package examples.rules.format;

public class TC {
    public static void main(String args[]) {
        System.out.println("Hello world!"); //TRAILING COMMENT, VIOLATION
    }
}
REPAIR 

package examples.rules.format;

public class TCFixed {
    public static void main(String args[]) {
        System.out.println("Hello world!");
        //NON-TRAILING COMMENT, FIXED
    }
}
REFERENCES
