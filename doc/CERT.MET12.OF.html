Do not overload the 'finalize()' method [CERT.MET12.OF]



DESCRIPTION 


This rule identifies 'finalize()' methods that are overloaded.
An error is reported for each occurrence.
SCOPE LEVEL 

METHOD
SINCE 

v6.0
BENEFITS 

A user-defined 'finalize()' method would be invoked during 
garbage collection only if it is overriden. An overloaded 
'finalize()' method that declares extra parameters would not
be executed by the VM.
EXAMPLE 

package examples.rules.pb.api;

public class OF {
    protected void finalize(Object o) { //VIOLATION
        super.finalize(); 
        //some other clean-ups
    }
}
REPAIR 

package examples.rules.pb.api;

public class OFFixed {
    protected void finalize() { //FIXED
        super.finalize(); 
        //some other clean-ups
    }
}
