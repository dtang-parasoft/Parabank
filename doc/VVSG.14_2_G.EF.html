Avoid empty "finalize()" methods [VVSG.14_2_G.EF]



DESCRIPTION 


This rule identifies code that declares a 'finalize()' method with an
empty body. An error is reported for each occurrence.
SCOPE LEVEL 

METHOD
SINCE 

v6.0
BENEFITS 

A 'finalize()' method with an empty body is meaningless. Consequently,
you should remove it to reduce code size.
EXAMPLE 

package examples.rules.uc;

public class EF {
    protected void finalize() throws Throwable {} //VIOLATION
}
REPAIR 

package examples.rules.uc;

public class EFFixed {
} //FIXED
REFERENCES 

SEI CERT Oracle Coding Standard for Java
MET12-J. Do not use finalizers
