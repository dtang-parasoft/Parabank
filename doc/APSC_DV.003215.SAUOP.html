Enforce number of space character(s) between a prefixed unary operator and its operand [APSC_DV.003215.SAUOP]



DESCRIPTION 

    
This rule identifies code that contains white space between a unary operator and its operand.
It applies to prefixed "++" and "--" operators. An error is reported for each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v4.5
PARAMETERS 

-Number of spaces after unary operators (default value is 0)
BENEFITS 

Space between a unary operator can make code confusing and error-prone.
EXAMPLE 

package examples.rules.format;

public class SAUOP {
    public void method (int x) {
        System.out.println("x: " + ++ x);  // VIOLATION
    }
   
}
REPAIR 

Remove the space.

package examples.rules.format;

public class SAUOPFixed {
    public void method (int x) {
        System.out.println("x: " + ++x);  // FIXED
    }
   
}
