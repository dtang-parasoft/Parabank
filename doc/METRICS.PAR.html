DEPRECATED: Number of parameters [METRICS.PAR]



DESCRIPTION 


Rules from this category are deprecated - see METRIC category.

Counts the number of method parameters.
NOTES 

Jtest will report an error if the value is greater or equal 7.
BENEFITS 

A high number of parameters indicate a complex interface to calling objects
and should be avoided.
REFERENCES 

By Mark Schroeder "A Practical Guide to Object-Oriented Metrics"
IT Pro, November/December, 1999.
