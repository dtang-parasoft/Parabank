A2-Broken Authentication [OWASP2017.A2]

RULES

Avoid using the 'getRequestedSessionId' method from the 'HttpServletRequest' class [OWASP2017.A2.HGRSI]
Do not rely on IP addresses obtained from HTTP request headers for authentication [OWASP2017.A2.HTTPRHA]
Always call 'HttpSession.invalidate()' before 'LoginContext.login()' [OWASP2017.A2.ISL]
Ensure proper session expiration [OWASP2017.A2.RUIM]
Ensure that sessions are configured to time out in 'web.xml' files [OWASP2017.A2.STTL]
Protect against using unprotected credentials [OWASP2017.A2.TDPASSWD]
