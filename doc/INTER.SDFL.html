Provide a 'Locale' argument when instantiating 'SimpleDateFormat' objects [INTER.SDFL]



DESCRIPTION 


This rule identifies code where a 'SimpleDateFormat' object is 
instantiated without a 'Locale' argument. An error is reported for 
each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v6.0
BENEFITS 

Because SimpleDateFormat is locale-sensitive, it is always
a good practice to specify the locale during instantiation
instead of assuming that the default locale is always suitable.
EXAMPLE 

package examples.rules.misc;

public class SDFL {
    SimpleDateFormat dateFormat= new SimpleDateFormat("MM/dd/yyyy"); //VIOLATION
}
REPAIR 

package examples.rules.misc;

public class SDFLFixed {
    SimpleDateFormat dateFormat= new SimpleDateFormat("MM/dd/yyyy", Locale.US); //FIXED
}
