Do not define custom class loaders [SECURITY.WSC.CL]



DESCRIPTION 


This rule identifies custom 'ClassLoader' declarations. A message
is reported for each occurrence.
SINCE 

v6.0
SECURITY RELEVANCE 

category: Weak Security Controls
category: Malicious Code

Creating a custom class loader that is not secure can allow
the loading of malicious classes.
EXAMPLE 

package examples.rules.security.wsc;

public class CL extends ClassLoader {} //VIOLATION
REPAIR 

Do not create custom 'ClassLoader' or ensure that 
custom ClassLoader is securely implemented.
