DEPRECATED: Number of package-private fields [METRICS.NPKGF]



DESCRIPTION 


Rules from this category are deprecated - see METRIC category.

The number of fields in a class indicates the amount of data
the class must maintain in order to carry out its responsibilities.
NOTES 

Jtest will report an error if the value is greater or equal 20.
