Place the 'main()' method last [CODSTA.ORG.PML]



DESCRIPTION 


This rule identifies classes whose last method is not the 'main()' method.
An error is reported if a class's 'main()' method is not its last method.
SCOPE LEVEL 
METHOD
SINCE 

v4.5
BENEFITS 

Following this rule helps enforce stylistic conventions across a project.
EXAMPLE 

package examples.rules.codsta.org;

public class PML {
    public static void main (String args[]) {  // VIOLATION
        System.out.println("Hello, world.");
    }
    public void foo () {
    }
}
REPAIR 

List the 'main()' method last within the "class" definition.

package examples.rules.codsta.org;

public class PMLFixed {
    public void foo () {
    }
    public static void main (String args[]) {  // FIXED
        System.out.println("Hello, world.");
    }
}
