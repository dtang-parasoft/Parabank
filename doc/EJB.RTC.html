Make the return type "void" for SessionBeans or MessageDrivenBeans' 'ejbCreate()' methods [EJB.RTC]



DESCRIPTION 


This rule identifies SessionBeans or MessageDrivenBeans whose 'ejbCreate ()' methods' 
do not have a "void" return type. An error is reported for each occurrence.
SCOPE LEVEL 

METHOD
SINCE 

v4.5
