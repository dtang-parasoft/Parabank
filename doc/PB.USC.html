Useless Code [PB.USC]

RULES

Avoid empty statements [PB.USC.AES]
Avoid conditional expressions that always evaluate to a constant value [PB.USC.CC]
Avoid empty classes [PB.USC.EC]
Do not define empty "public" constructors in classes with no other constructors [PB.USC.EPC]
Avoid using "private" fields which are never given a meaningful value [PB.USC.FCBS]
Avoid classes with no accessible members [PB.USC.NACC]
Ensure method and constructor return values are used [PB.USC.NASSIG]
Avoid increment and decrement statements which have no effect [PB.USC.OI]
Do not catch exceptions only to rethrow them [PB.USC.RTE]
Avoid self assignments/initializations to fields and/or local variables [PB.USC.SAFL]
Avoid unreachable "else if" and "else" cases [PB.USC.UIF]
Do not use the unary operator '+' [PB.USC.UNARY]
