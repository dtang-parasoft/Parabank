Put a blank line before each C-style comment [FORMAT.BLBC]



DESCRIPTION 


Every block (C-style) comment should have a blank line preceding it.  This rule
will flag any block comment which is not immediately preceded by a blank line.
SCOPE LEVEL 
LINE
SINCE 

v8.0
BENEFITS 

Placing a blank line before a block comment helps to set the comment apart from
the rest of the code.  This improves the readability of the code.
EXAMPLE 

package examples.rules.format;

public class BLBC{
    public void myMeth(){
        int i= 0;
        /*      //VIOLATION- no blank line preceding comment
         * increments i
         */
        i= i+1;
    }
}
REPAIR 

Place a blank line before the comment.

public class BLBCFixed{
    public void myMeth(){
        int i= 0;
                //FIXED
        /*    
         * increments i
         */
        i= i+1;
    }
}
