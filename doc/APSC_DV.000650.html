APSC_DV.000650 The application must not write sensitive data into the application logs. [APSC_DV.000650]

RULES

Do not log confidential or sensitive information [APSC_DV.000650.CONSEN]
Ensure all sensitive method invocations are logged [APSC_DV.000650.ENFL]
Avoid logging sensitive Hibernate-related information at the 'info' level in 'log4j.properties' files [APSC_DV.000650.LHII]
Prevent exposure of sensitive data [APSC_DV.000650.SENS]
Avoid passing sensitive data to functions that write to log files [APSC_DV.000650.SENSLOG]
