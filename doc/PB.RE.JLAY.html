Avoid using methods add() from JLayer class [PB.RE.JLAY]



DESCRIPTION 

JLayer is a decorator for Swing components that allows advanced painting effects.
Although it inherits class java.awt.Component, JLayer does not support the following methods: 
    
    Container.add(java.awt.Component)
    Container.add(String, java.awt.Component)
    Container.add(java.awt.Component, int)
    Container.add(java.awt.Component, Object)
    Container.add(java.awt.Component, Object, int)

Using any of them will cause UnsupportedOperationException to be thrown.

For such cases, this rule reports a violation.
SINCE 
v9.4
BENEFITS 
setView(Component) or setGlassPane(JPanel) methods must be used to add a component to JLayer.
Mistakes can easily be made, though, because JLayer inherits java.awt.Component.
This rule is in place to prevent this problem.
EXAMPLE 

    public void test() {

        JButton jButton = new JButton();

        JFrame jFrame = new JFrame();

        JPanel jPanel = new JPanel();
        LayerUI<JPanel> layerUI = new LayerUI<JPanel>();
        JLayer<JPanel> jLayer = new JLayer<JPanel>(jPanel, layerUI);
        jLayer.add(jButton); //VIOLATION

        jFrame.add(jLayer);
    }
REPAIR 

    public void test() {

        JButton jButton = new JButton();

        JFrame jFrame = new JFrame();

        JPanel jPanel = new JPanel();
        LayerUI<JPanel> layerUI = new LayerUI<JPanel>();
        JLayer<JPanel> jLayer = new JLayer<JPanel>(jPanel, layerUI);
        // removed jLayer.add(jButton); FIXED

        jFrame.add(jLayer);
    }
