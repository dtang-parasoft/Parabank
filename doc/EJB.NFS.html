Declare all "static" fields in EJB bean classes "final" [EJB.NFS]



DESCRIPTION 


This rule identifies "static" fields that are not declared "final" in 
"SessionBean" or "EntityBean" classes. An error is reported for each
occurrence.
SINCE 

v4.5
BENEFITS 

This ensures consistent runtime semantics so that EJB containers have
the flexibility to distribute instances across multiple JVMs.

Enterprise JavaBeans Programming restrictions on EJB
