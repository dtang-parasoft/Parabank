Ensure that Secure Processing is used [SECURITY.WSC.FXMLP]



DESCRIPTION 


This rule identifies instantiations of certain xml text builders that do not 
explicitly set the XMLConstants.FEATURE_SECURE_PROCESSING property to true.
The specified classes can be changed in the parameters.
SINCE 

v9.0
SECURITY RELEVANCE 

See benefits section.
PARAMETERS 

-List of classes to check
BENEFITS 

XMLConstants.FEATURE_SECURE_PROCESSING can prevent DDOS attacks and set limits
on XML constructs.
EXAMPLE 

package examples.rules.security.wsc;

import javax.xml.parsers.DocumentBuilderFactory;

public class FXMLP {
    private void method() {
        DocumentBuilderFactory dbf = DocumentBuilderFactory.newInstance();//VIOLATION
    }
}
REPAIR 

package examples.rules.security.wsc;

import javax.xml.parsers.DocumentBuilderFactory;
import javax.xml.parsers.ParserConfigurationException;

     public class FXMLPFixed {
        private void method() throws ParserConfigurationException {
            DocumentBuilderFactory dbf = DocumentBuilderFactory.newInstance();
            dbf.setFeature("http://javax.xml.xmlconstants/feature/secure-processing",true);//FIXED
        }
    }
