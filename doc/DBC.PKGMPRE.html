Provide a '@pre' contract for all package-private methods [DBC.PKGMPRE]



DESCRIPTION 


This rule identifies any package-private method without a '@pre' contract.
An error is reported for each occurrence.
SCOPE LEVEL 

METHOD
SINCE 

v4.5
NOTES 

If there's no Javadoc comment, Jtest will not report this error.
BENEFITS 

Pre-conditions check that the client calls the method correctly.
EXAMPLE 

package examples.rules.dbc;

public class PKGMPRE
{
    /**
     * method setMonth()
     */
    void setMonth (int month) {  // VIOLATION
        _month = month;
    }
    private int _month;
}
REPAIR 

Provide a '@pre' contract, or use '/** @pre $none */' to document the lack of one.

package examples.rules.dbc;

public class PKGMPREFixed
{
    /** @pre month >= 1 && month <= 12 */  // FIXED
    void setMonth (int month) {
        _month = month;
    }
    private int _month;
}
