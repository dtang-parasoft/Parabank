Avoid empty static and non-static initializers [VVSG.14_2_G.AESTAT]



DESCRIPTION 


This rule identifies empty static or non-static initializers. An error is
reported for each occurrence.
SINCE 

v6.0
PARAMETERS 

-Check static initializers (default true)
-Check non-static initializers (default false)
BENEFITS 

Empty static and non-static initializers are meaningless. They should be removed
to simplify code.
EXAMPLE 

package examples.rules.uc;

public class AESTAT {
    static {   //VIOLATION
    }
}
REPAIR 

package examples.rules.uc;

public class AESTATFixed {
    //FIXED, empty static initializer is removed
}
