Global Static Analysis [GLOBAL] 

Rules that perform Global Static Analysis.

Jtest first scans all the classes in the
project to collect global usage information.
When testing each class, it uses that
information to check the rules in this category.

Note that by default, user-defined test cases are not
considered in this analysis. To configure global
static analysis to check JUnit test classes, enable
the following Test Configuration option:
Static> Options> Include user-defined test
classes in global analysis.
RULES

Ensure that files do not contain cyclical dependencies [GLOBAL.ACD]
Make methods "static" if they do not use instance class members [GLOBAL.ASI]
Avoid unused "throws" clauses [GLOBAL.AUT]
Declare "package-private" types as inaccessible as possible [GLOBAL.DPAC]
Declare package-private fields as inaccessible as possible [GLOBAL.DPAF]
Declare package-private methods as inaccessible as possible [GLOBAL.DPAM]
Declare "public/protected" types as inaccessible as possible [GLOBAL.DPPC]
Declare "public/protected" fields as inaccessible as possible [GLOBAL.DPPF]
Declare "public/protected" methods as inaccessible as possible [GLOBAL.DPPM]
Avoid interfaces which are never implemented and "abstract" classes which are never extended [GLOBAL.NIE]
Declare non-private fields "final" if they are read-only [GLOBAL.SF]
Declare a package-private "class" "final" if it is not subclassed [GLOBAL.SPAC]
Declare a package-private method "final" if it is not overridden [GLOBAL.SPAM]
Declare a "public/protected" "class" "final" if it is not subclassed [GLOBAL.SPPC]
Declare a "public/protected" method "final" if it is not overridden [GLOBAL.SPPM]
Ensure correct constructor declarations in utility classes [GLOBAL.UCC]
Avoid globally unused enum constants [GLOBAL.UEC]
Avoid globally unused package-private types [GLOBAL.UPAC]
Avoid globally unused package-private fields [GLOBAL.UPAF]
Avoid globally unused package-private methods [GLOBAL.UPAM]
Avoid globally unused "public/protected" types [GLOBAL.UPPC]
Avoid globally unused "public/protected" fields [GLOBAL.UPPF]
Avoid globally unused "public/protected" methods [GLOBAL.UPPM]
