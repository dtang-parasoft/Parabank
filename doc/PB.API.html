API Usage and Implementation [PB.API]

RULES

Avoid extending certain classes (custom rule) [PB.API.AECC]
Avoid 'put, 'putAll' and 'get' methods of 'java.util.Properties' objects [PB.API.APPG]
Avoid using certain packages or types [PB.API.APT]
Ensure that the types passed to "Collection" and "Map" methods match the types in those data structures [PB.API.CMMT]
Add the "@Deprecated" annotation to the declarations of deprecated members [PB.API.DANNOT]
Do not call 'setSize()' in 'ComponentListener.componentResized()' [PB.API.DNCSS]
Do not use deprecated APIs [PB.API.DPRAPI]
Do not extend "Collection" and "Map" classes [PB.API.ECMC]
Do not extend 'java.util.HashMap' or 'java.util.Hashtable' [PB.API.EHM]
Do not create an empty JarFile entry or an empty ZipFile entry [PB.API.EJF]
Avoid calling 'equals(null)' [PB.API.EQNL]
Do not call the 'next' method from the 'hasNext' method of an Iterator [PB.API.HNCN]
Do not use "URL" objects in "Collections" or "Maps" [PB.API.IUMS]
Ensure that all types which are used as keys in Sets and Maps override the 'equals()' and 'hashCode()' methods [PB.API.KOEH]
Assign 'protected' accessibility to 'readResolve()' and 'writeReplace()' methods in serializable classes [PB.API.MASP]
Avoid using objects with dangerous implementations of 'equals()' or 'hashCode()' as keys in hashed data structures [PB.API.MOHK]
Do not define direct or indirect subclasses of 'Error' and 'Throwable' [PB.API.NDC]
Do not overload the 'finalize()' method [PB.API.OF]
Ensure method arguments are serializable [PB.API.ONS]
Avoid using "." as a regular expression in 'String.replaceAll()' and 'String.replaceFirst()' [PB.API.REP]
Do not pass a 'char' to the 'StringBuffer(int)' constructor [PB.API.SBCC]
Avoid improper casting of the results of the 'next' methods in the 'java.util.Random' class [PB.API.UNI]
Avoid using the 'equals()' and 'hashCode()' methods of "java.net.URL" [PB.API.URL]
Ensure the correct number of arguments for varargs methods with format strings [PB.API.VAFS]
Update values which may be using to compare with vendor properties [PB.API.VENDOR]
