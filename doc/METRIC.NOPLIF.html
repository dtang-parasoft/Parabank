Physical Lines in Files [METRIC.NOPLIF]



DESCRIPTION 


Number of Physical Lines metrics measures the total number of lines. It covers all
kinds of lines: source code lines, lines with comments, and blank lines.
This metric measures the number of physical lines in file.
