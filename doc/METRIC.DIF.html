Depth of Nested 'if' Statements [METRIC.DIF]



DESCRIPTION 

This metric calculates the maximum depth of if blocks in a method.
It assumes that the depth of the initial block (the first set of
curly brackets) is 0, and nested if blocks increase the value.
SINCE 

v10.3.0
BENEFITS 
A large number of nested if statements makes code difficult to understand
and maintain.
EXAMPLE 

        public void test2(){
            for(;;){
                if(true){//1
                    if(true){//2
                        switch(a){
                        case 1:
                            while(true){
                                if(true){//3
                                    if(true){//4
                                        
                                    }
                                }
                            }
                        case 2:
                            if(true){//3
                                while(true){
                                    if(true){//4	
                                    }
                                    else{
                                        if(true){//5	
                                        }
                                    }
                                }
                            }
                        }
                    }
                }
            }
        }
}
Max: 5
