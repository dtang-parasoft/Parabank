Do not import classes from the package that contains the current class [VVSG.14_2_G.PIMPORT]



DESCRIPTION 


This rule identifies any class that imports other classes from that same 
package. An error is reported for each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v6.0
BENEFITS 

When a class is in a specific package, that class has access to other
classes inside the same package. Consequently, it is redundant to import those classes 
again.
EXAMPLE 

package examples.rules.uc;

import examples.rules.uc.*; // VIOLATION
import examples.rules.uc.DIMPORT; //VIOLATION

public class PIMPORT {
}
REPAIR 

package examples.rules.uc;

//FIXED

public class PIMPORTFixed {
}
