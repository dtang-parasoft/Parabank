Declare a "public" constructor that takes no parameters in bean classes [EJB.NFDC]



DESCRIPTION 


This rule identifies any "MessageDrivenBean", "EntityBean", or "SessionBean" 
class that does not have a "public" constructor that takes no parameters.  An 
error is reported for each occurrence.
SCOPE LEVEL 

CLASS
SINCE 

v4.5
BENEFITS 

The container uses the no-argument constructor to create an instance of the 
bean class.
REFERENCES 

Sections 7.10.2 (p. 95) and 10.5.2 (p. 170), and 15.7.2 (p. 323) of the EJB 
specification (version 2.0).  The EJB specification is available online at 
http://download.oracle.com/otndocs/jcp/7294-ejb-2.0-fr2-spec-oth-JSpec/
