DEPRECATED: "class" or "interface" inheritance level [METRICS.ID]



DESCRIPTION 


Rules from this category are deprecated - see METRIC category.

Measures the inheritance level of a "class".
NOTES 

Jtest will report an error if the value is greater or equal 7.
BENEFITS 

An unnecessarily deep "class" hierarchy adds to complexity and can represent a
poor use of the inheritance mechanism.
REFERENCES 

By Mark Schroeder "A Practical Guide to Object-Oriented Metrics"
IT Pro, November/December, 1999.
