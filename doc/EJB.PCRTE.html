Declare 'ejbPostCreate()' "public" and neither "static" nor "final" [EJB.PCRTE]



DESCRIPTION 


This rule identifies 'ejbPostCreate ()' methods that are either
"final", "static" or non-"public". An error is reported
for each occurrence.
