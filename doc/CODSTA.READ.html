Readability [CODSTA.READ]

RULES

Do not rely on automatic boxing and unboxing of primitive types [CODSTA.READ.ABUB]
Do not use complicated conditional expressions in control structures [CODSTA.READ.ACCS]
Avoid or enforce usage of enhanced "for" loops [CODSTA.READ.AEFS]
Access and set fields directly in the declaring type instead of using getter and setter methods [CODSTA.READ.AFD]
Do not use fully qualified type names [CODSTA.READ.AFQN]
Avoid anonymous inner classes [CODSTA.READ.AIC]
Avoid using negative logic in if-else statement [CODSTA.READ.ANL]
Avoid static import statements [CODSTA.READ.ASIS]
Avoid unnecessary calls to 'toString()' [CODSTA.READ.AUTS]
Comment the ends of control structures [CODSTA.READ.CCB]
Comment empty blocks [CODSTA.READ.CEB]
Avoid using increment or decrement operators in nested expressions [CODSTA.READ.CID]
Comment local variables [CODSTA.READ.CLV]
Always call 'Collection.toArray()' with an empty constant array argument [CODSTA.READ.CTA]
Enforce or avoid usage of conditional operators [CODSTA.READ.CX]
Avoid using "do-while" statements [CODSTA.READ.DOWHILE]
Do not use too many negation operators '!' in a single method [CODSTA.READ.DUN]
Declare variables as close as possible to where they are used [CODSTA.READ.DVCU]
Explicitly call one of the superclass' constructors from all constructors [CODSTA.READ.ECSC]
Declare "private" constant fields "final" [CODSTA.READ.FF]
Declare constant local variables "final" [CODSTA.READ.FLV]
Avoid using Hexadecimal binary exponents [CODSTA.READ.HBE]
Avoid having a lower-case "l" or the number "1" at the end of a "long" integer constant [CODSTA.READ.LONG]
Minimize "try" block size [CODSTA.READ.MTBS]
Do not declare multiple variables in one statement [CODSTA.READ.MVOS]
Avoid nested assignments or assignments embedded in other expressions [CODSTA.READ.NEA]
Declare never-modified collections as "unmodifiable" for clarity [CODSTA.READ.NMUC]
Avoid non-static initializers [CODSTA.READ.NSI]
Declare "for" loops with an initializer, conditional, and updater statements [CODSTA.READ.PCIF]
Do not declare "public" constructors in non-public classes [CODSTA.READ.PCTOR]
Put declarations only at the beginning of blocks [CODSTA.READ.PDBB]
Enforce use of "for" or "while" loops [CODSTA.READ.PFL]
Use 'isEmpty()' for Collections and Maps instead of comparing 'size()' to 0 [CODSTA.READ.SIE]
Avoid or enforce the use of "this" and "super" expressions [CODSTA.READ.UATS]
Avoid create integer values from binary notation using Integer.parseInt() methods [CODSTA.READ.UBL]
Use underscore characters (_) in numerical literal [CODSTA.READ.ULIT]
Avoid literal constants [CODSTA.READ.USN]
Do not declare multiple variables of different types in one statement [CODSTA.READ.VDT]
Limit the number of initialization and update statements in "for" loops [CODSTA.READ.VIFS]
