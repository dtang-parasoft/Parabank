Do not explicitly "import" the java.lang.* "package" [UC.DIL]



DESCRIPTION 


This rule identifies code that explicitly imports the 'java.lang' "package". An
error is reported for each occurrence.
SCOPE LEVEL 

FILE
SINCE 

v4.5
BENEFITS 

It is not necessary to "import" this "package" because it is implicitly imported.
EXAMPLE 

package examples.rules.uc;

import java.lang.*;  // VIOLATION

public class DIL {
}
REPAIR 

Remove the statement that imports 'java.lang'.

package examples.rules.uc;

// FIXED

public class DILFixed {
}
