Do not start business method names with "ejb" [EJB.EJB3.BMN]



DESCRIPTION 


This rule flags a violation if a method in a business interface starts with
"ejb". Methods should not use this formatting to avoid conflicts with callback
methods.
SINCE 

v8.4
NOTES 

An interface is considered a business interface if it implements one of the
following annotations: @Local, @Remote, @WebService.
EXAMPLE 

package examples.rules.ejb.ejb3;

@Local
public interface BMN {
    void ejbMethod(); //VIOLATION
    ...
}
REPAIR 

Rename the method.

package examples.rules.ejb.ejb3;

@Local
public interface BMNFixed {
    void method(); //Fixed
    ...
}
REFERENCES 

Debu Panda, Reza Rahman, and Derek Lane. EJB3 in Action.
Greenwich: Manning, 2007. p. 78.
