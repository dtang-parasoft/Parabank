Enforce number of space character(s) before and after the "?" conditional operator [APSC_DV.003215.SCOP]



DESCRIPTION 

   
This rule identifies any "?" conditional operator that is not preceded and 
followed by the required number of space characters. With the default parameters, 
an error is reported when a "?" conditional operator is not preceded and followed
by one space character.
SCOPE LEVEL 
LINE
SINCE 

v4.5
PARAMETERS 

-Number of spaces before conditional operators (default value is 1)
-Number of spaces after conditional operators (default value is 1)
BENEFITS 

If all team members follow the same formatting style, it will be easier for each
team member to read and understand the code.
EXAMPLE 

package examples.rules.format;

public class SCOP {
    public int method (int i) {
        int j = (i < 0)? 0 : i;  // VIOLATION
        return j;
    }
}
REPAIR 

Place a single space before and after the "?" operator.

package examples.rules.format;

public class SCOPFixed {
    public int method (int i) {
        int j = (i < 0) ? 0 : i;  // FIXED
        return j;
    }
}
