Do not use 'StringTokenizer' in an Internationalized environment [INTER.ST]



DESCRIPTION 


This rule identifies the use of 'StringTokenizer()'. An error is reported for 
each occurrence.
SINCE 

v4.5
EXAMPLE 

package examples.rules.inter;

import java.util.*;

public class ST{
    public int method (String str) {
        StringTokenizer st = new StringTokenizer(str);  // VIOLATION
        return st.countTokens ();
    }
}
REPAIR 
    
Please see the reference for detailed guidance on using Strings in an
internationalized environment.
REFERENCES 

http://java.sun.com/docs/books/tutorial/i18n/intro/checklist.html
