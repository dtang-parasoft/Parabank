DESCRIPTION 


Number of Logical Lines metrics measures the number of logical lines of source
code. Logical lines are sometimes called statements. One logical line may span over
several physical lines of code or there may be many logical lines in one physical line
of code. This metric counts the number of logical lines that belong to types.
