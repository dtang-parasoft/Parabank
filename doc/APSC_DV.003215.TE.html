Use the preferred formatting for conditional expressions [APSC_DV.003215.TE]



DESCRIPTION 


This rule identifies conditional expressions (a ? b : c) that do not use 
the preferred formatting. An error is reported if the expressions do not follow
these three acceptable formats:
 
alpha = (aLongBooleanExpression) ? beta : gamma;  
alpha = (aLongBooleanExpression) ? beta
                                 : gamma;  
alpha = (aLongBooleanExpression)
        ? beta 
        : gamma;  

When an expression will not fit on a single line, break it according to
general principles.
SCOPE LEVEL 
LINE
SINCE 

v5.0
PARAMETERS 

-Enforced formats for conditional expressions:
    -expression1 ? expression2 : expression3     (default is true)
    -expression1 ? expression2                   (default is true)
                 : expression3
    -expression1                                 (default is true)
     ? expression2
     : expression3
-The condition must be parenthesized (default is false)
    -If this parameter is enabled, a violation will be flagged if the 
     condition part of the conditional expression is not parenthesized.
BENEFITS 

If all team members follow the same formatting style, it will be easier for each
team member to read and understand the code.
EXAMPLE 

package examples.rules.format;

class TE {
    void foo (int radix, String str, String count) {
        int size = str.endsWith ("str") ? Integer.parseInt (count) : 
            Integer.parseInt (count, radix); //VIOLATION           
        System.out.println ("size = " +size);
    }
}
REPAIR 

Modify the code to follow the coding standard.

package examples.rules.format;

class TEFixed {
    void foo (int radix, String str, String count) {
        int size = str.endsWith ("str") 
                   ? Integer.parseInt (count) 
                   : Integer.parseInt (count, radix);           
        /* OR this
        int size = str.endsWith ("str") ? Integer.parseInt (count) 
                                        : Integer.parseInt (count, radix);
        */
        System.out.println ("size = " +size);
    }
}
