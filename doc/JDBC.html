Java Database Connectivity [JDBC] 

Rules related to Java Database Connectivity (JDBC).
RULES

Ensure index is valid in JDBC method invocation [JDBC.BRSA]
Close JDBC connections in "finally" blocks [JDBC.CDBC]
Close JDBC objects in the correct order [JDBC.COCO]
Reuse data sources for JDBC connections [JDBC.DSLV]
Do not open or close JDBC connections in loops [JDBC.ODBIL]
Close JDBC resources in "finally" blocks [JDBC.RRWD]
Do not store database connection objects in "static" fields [JDBC.SCSF]
Use javax.sql.DataSource to get the database connection [JDBC.UDS]
Use "PreparedStatements" correctly [JDBC.UPSC]
Use instance of RowSetFactory to create a RowSet object [JDBC.URSF]
