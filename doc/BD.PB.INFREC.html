Avoid infinite recursion [BD.PB.INFREC]



DESCRIPTION 

This rule detects infinitely recursive methods. A method is infinitely recursive when every
path that results from the method being entered leads to a recursive call. This rule also
detects indirect recursion.
SINCE 
v10.0.8
NOTES 
N/A
SECURITY RELEVANCE 
N/A
PARAMETERS 
N/A
BENEFITS 
N/A
DRAWBACKS 
N/A
EXAMPLE 
Here is an example that triggers a violation:

public class Example
{
    public int binarySearch(int[] tab, int a, int b, int val) // VIOLATION
    {
        int index = (a + b) / 2;
        if (tab[index] > val) {
            return binarySearch(tab, a, index - 1, val);
        }
        return binarySearch(tab, index + 1, b, val);
    }
}
REPAIR 

public class Repair
{
    public int binarySearch(int[] tab, int a, int b, int val) // NO VIOLATION
    {
        int index = (a + b) / 2;
        if (tab[index] > val) {
            return binarySearch(tab, a, index - 1, val);
        } else if (tab[index] < val) {
            return binarySearch(tab, index + 1, b, val);
        } else {
            return index;
        }
    }
}
