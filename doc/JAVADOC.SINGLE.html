Enforce restraint on number of lines used for Javadoc comments [JAVADOC.SINGLE]



DESCRIPTION 


This rule would report an error if the number of lines used
in the javadoc comment does not follow the setting.
SCOPE LEVEL 
LINE
SINCE 

v6.0
PARAMETERS 

-Javadoc format option:
    -Always use multiple lines for javadoc comments (default option)
    -Always use single line for javadoc comments if they fit
EXAMPLE 

package examples.rules.JAVADOC;

public class SINGLE {
    /**Void method that does nothing*/    //VIOLATION
    public void myMethod() {}
}
REPAIR 

package examples.rules.JAVADOC;

public class SINGLE {
    /** 
     * Void method that does nothing //FIXED
     */   
    public void myMethod() {} 
}
