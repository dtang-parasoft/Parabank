2.5.3-A: Escaping and encoding output [VVSG.2_5_3_A]

RULES

Do not call the 'printStackTrace()' method of "Throwable" objects [VVSG.2_5_3_A.ACPST]
Avoid debug information from Spring Security framework to logs [VVSG.2_5_3_A.EWSSEC]
Do not pass exception messages into output in order to prevent the application from leaking sensitive information [VVSG.2_5_3_A.PEO]
Avoid calling print methods of 'System.err' or 'System.out' [VVSG.2_5_3_A.SIO]
