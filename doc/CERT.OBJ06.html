OBJ06-J: Defensively copy mutable inputs and mutable internal components [CERT.OBJ06]

RULES

Enforce returning a defensive copy in 'clone()' methods [CERT.OBJ06.CPCL]
Do not pass user-given mutable objects directly to certain types [CERT.OBJ06.MPT]
Provide mutable classes with copy functionality [CERT.OBJ06.MUCOP]
Do not store user-given mutable objects directly into variables [CERT.OBJ06.SMO]
