A8-Insecure Deserialization [OWASP2017.A8]

RULES

Avoid parsing untrusted data with XMLDecoder [OWASP2017.A8.AUXD]
Create a 'serialVersionUID' for all 'Serializable' classes [OWASP2017.A8.DUID]
Implement 'readObject()' and 'writeObject()' for all 'Serializable' classes [OWASP2017.A8.OROM]
Define a "readResolve" method for all instances of Serializable types [OWASP2017.A8.RRSC]
Ensure that all fields are assigned by the 'readObject()' method and written out by the 'writeObject()' method [OWASP2017.A8.RWAF]
Disable LDAP deserialization [OWASP2017.A8.SC]
Always declare writeObject and readObject methods for Serializable subclasses of non-Serializable parents [OWASP2017.A8.SCBNP]
Do not declare the 'readObject()' method as "synchronized" [OWASP2017.A8.SROS]
Validate objects before deserialization [OWASP2017.A8.VOBD]
