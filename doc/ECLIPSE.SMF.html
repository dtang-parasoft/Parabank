Avoid missing "Eclipse-AutoStart" entry in MANIFEST.MF [ECLIPSE.SMF]



DESCRIPTION 


This rule identifies instances when Eclipse-AutoStart is missing in MANIFEST.MF.
An error is reported for each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v8.1
EXAMPLE 

# .....
Bundle-Activator: com.parasoft.jtest.Plugin
# .....
# VIOLATION : missing 'Eclipse-AutoStart'
REPAIR 

# .....
Bundle-Activator: com.parasoft.jtest.Plugin
# .....
# FIXED : add 'Eclipse-AutoStart'
Eclipse-AutoStart: true
