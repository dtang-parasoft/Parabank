Avoid 'public' or 'protected' constructors for immutable classes [CODSTA.BP.CMUTA]



DESCRIPTION 


This rule identifies immutable classes with constructors that are public or 
protected. A violation is reported for each constructor.
SINCE 

v9.0
BENEFITS 

Hiding constructors allows more flexibility in instance creation and caching.
Creating build methods to construct these objects is a good practice.
EXAMPLE 

package examples.rules.codsta.bp;

public class CMUTA {
    private int a;
    
    public CMUTA(){ //VIOLATION
    }
    
    private void method() {
    }
}
REPAIR 

package examples.rules.codsta.bp;

public class CMUTAFixed {
    private int a;
    
    private CMUTA(){ //FIX
    }
    
    private void method() {
    }
}
