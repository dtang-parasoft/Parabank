6.5.4 Insecure communications [PCIDSS32.654]

RULES

Do not log confidential or sensitive information [PCIDSS32.654.CONSEN]
Avoid using hard-coded cryptographic keys [PCIDSS32.654.HCCK]
Use 'https' instead of 'http' for the 'transportReceiver' and 'transportSender' in 'axis2.xml' configuration files [PCIDSS32.654.HTTPS]
MessageDigest objects must process the data with the 'update' method [PCIDSS32.654.MCMDU]
Initialize KeyGenerator instances [PCIDSS32.654.SIKG]
Use the SSL-enabled version of classes when possible [PCIDSS32.654.USC]
