Declare finder methods "public" and neither "final" nor "static" [EJB.FNDM]



DESCRIPTION 


This rule identifies Finder methods that are either "final", "static"
or non-"public". An error is reported for each occurrence.
SINCE 

v4.5
EXAMPLE 

package examples.rules.ejb;

import java.rmi.RemoteException;
import javax.ejb.*;

public class FNDM implements EntityBean {
    static void ejbFindByCreate(int i){  //VIOLATION
        //Method Stub
    }
    
    //Stubs for EntityBean methods
    public void setEntityContext(EntityContext arg0) throws EJBException, RemoteException { }
    public void unsetEntityContext() throws EJBException, RemoteException { }
    public void ejbRemove() throws RemoveException, EJBException, RemoteException { }
    public void ejbActivate() throws EJBException, RemoteException { }
    public void ejbPassivate() throws EJBException, RemoteException { }
    public void ejbLoad() throws EJBException, RemoteException { }
    public void ejbStore() throws EJBException, RemoteException { }
    public void ejbCreate() throws EJBException { }
}
REPAIR 

Declare the finder method as "public", non-"final", and non-"static".

package examples.rules.ejb;

import java.rmi.RemoteException;
import javax.ejb.*;

public class FNDMFixed implements EntityBean {
    public void ejbFindByCreate(int i){  //FIXED  
        //Method Stub
    }
    
    //Stubs for EntityBean methods
    public void setEntityContext(EntityContext arg0) throws EJBException, RemoteException { }
    public void unsetEntityContext() throws EJBException, RemoteException { }
    public void ejbRemove() throws RemoveException, EJBException, RemoteException { }
    public void ejbActivate() throws EJBException, RemoteException { }
    public void ejbPassivate() throws EJBException, RemoteException { }
    public void ejbLoad() throws EJBException, RemoteException { }
    public void ejbStore() throws EJBException, RemoteException { }
    public void ejbCreate() throws EJBException { }    
}
