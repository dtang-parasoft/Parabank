Use the '@version' tag in type Javadoc comments [JAVADOC.MVJDT]



DESCRIPTION 


This rule identifies classes, interface, enum, annotation that have Javadoc comments but 
lack the '@version' Javadoc tag. An error is reported for each occurrence.
SINCE 

v4.5
EXAMPLE 

package examples.rules.javadoc;

/**
 * class MVJDT
 */
public class MVJDT {  // VIOLATION
    void method() {}
}
REPAIR 

Provide the '@version' tag.

package examples.rules.javadoc;

/** @version 1.0
 */
public class MVJDTFixed {  // FIXED
    void method() {}
}
