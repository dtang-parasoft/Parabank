Metrics - deprecated [METRICS] 

Rules from this category are deprecated - see METRIC category.

Rules that measure metrics information for classes and methods.

Jtest reports a rule violation if any metrics measured are
outside of the specified "legal" bounds. 

To edit the upper and lower thresholds for a metric rule you have to edit rule using RuleWizard.
