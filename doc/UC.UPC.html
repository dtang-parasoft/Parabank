Avoid unused "private" classes or interfaces [UC.UPC]



DESCRIPTION 


This rule identifies code that defines unused "private" types.  An error is
reported for each occurrence.
SCOPE LEVEL 
CLASS
SINCE 

v5.0
SECURITY RELEVANCE 

category: Weak Security Controls
category: Code Quality

The code may implement features that bypass security.
BENEFITS 

These unused entities usually point to either:

1) Old code that is no longer needed.  This makes the code more difficult to
   understand.

2) A logical flaw (if the entity needs to be used, but other classes incorrectly
   avoid using it).
EXAMPLE 

package examples.rules.uc;

public class UPC {

    /**
     * This is a test method
     */
    public void method() {

    }

    private class UPC_Inner { // VIOLATION : unused

    }
}
REPAIR 

Remove class 'UPC_Inner'.

package examples.rules.uc;

public class UPCFixed {

    /**
     * This is a test method
     */
    public void method() {

    }

    // FIXED : removed class UPC_Inner
}
