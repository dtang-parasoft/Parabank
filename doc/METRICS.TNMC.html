DEPRECATED: Number of method calls [METRICS.TNMC]



DESCRIPTION 


Rules from this category are deprecated - see METRIC category.

Counts calls to methods and system functions within a system, class, or method.
NOTES 

This rule does not test "abstract", "native", or any methods 
that are declared in the interface.

Jtest will report an error if the value is greater or equal 40.
BENEFITS 

Many method calls can indicate that a method is doing a lot of work and should
be separated into several smaller methods.
REFERENCES 

By Mark Schroeder "A Practical Guide to Object-Oriented Metrics"
IT Pro, November/December, 1999.
