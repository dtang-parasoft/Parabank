Do not catch the 'java.lang.Error' object [EXCEPT.NCERR]



DESCRIPTION 


This rule identifies any code that attempts to catch the 'java.lang.Error' 
object. An error is reported for each occurrence.
SINCE 

V6.0
BENEFITS 

The definition of 'java.lang.Error' is very broad. Consequently,  
if code catches 'java.lang.Error', it might catch some unexpected 
errors without dealing with them properly.
EXAMPLE 

package examples.rules.except;

public class NCERR {
    public void myMethod() {
        try {
            //...
	} catch (Error e) { //VIOLATION
        }
    
    }
}
REPAIR 

package examples.rules.except;

public class NCERRFixed {
    public void myMethod() {
	try {
            //...
	} catch (AssertionError e) { //FIXED
        }
    }
}
