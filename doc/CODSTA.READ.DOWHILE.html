Avoid using "do-while" statements [CODSTA.READ.DOWHILE]



DESCRIPTION 


This rule identifies code that uses "do-while" statement.
An error is reported for each occurence.
SINCE 

v6.0
BENEFITS 

Consider that the programmer looking at code is probably 
examining each method starting at the top and working down. 
When encountering a loop, the first thing the programmer 
wants to know is what terminates the loop. It makes it more
difficult to read if logic is located at the bottom.
EXAMPLE 

package examples.rules.codsta.read;

public class DOWHILE {
    public static void main (String[] args) {
        String name= null;
        do {   //VIOLATION
            System.out.println("Please enter a name:");
            name= getNameFromInput();
        } while (name== null); 
    }
}
REPAIR 

Restructure the "do-while" loop into other conditional 
loop if possible

package examples.rules.codsta.read;

public class DOWHILEFixed {
    public static void main (String[] args) {
        String name= null;
	while (name== null) { //fixed
            System.out.println("Please enter a name:");
            name= getNameFromInput();
        } 
    }
}
