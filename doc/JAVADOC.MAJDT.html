Use the '@author' Javadoc tag in declaration Javadoc comments [JAVADOC.MAJDT]



DESCRIPTION 


This rule identifies classes, interfaces, methods, fields, enums, annotations that lack a '@author' Javadoc tag. 
An error is reported for each occurrence.
SCOPE LEVEL 
CLASS
SINCE 

v4.5
PARAMETERS 

-Check type declarations (default is true)
-Check method/annotation member type declarations (default is false)
-Check field declarations (default is false)
EXAMPLE 

package examples.rules.javadoc;

/** 
  * @version:
  * @see:
  */  // VIOLATION
public class MAJDT {
    public void method() {}
}
REPAIR 

Provide the '@author' tag in Javadoc comments.

package examples.rules.javadoc;

/** 
  * @version:
  * @see:
  * @author Parasoft  // FIXED
  */
public class MAJDTFixed {
    public void method() {}
}
