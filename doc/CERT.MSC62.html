MSC62-J: Store passwords using a hash function [CERT.MSC62]

RULES

Avoid storing usernames and passwords in plain text in Castor 'jdo-conf.xml' files [CERT.MSC62.PCCF]
Password information should not be included in properties file in plaintext [CERT.MSC62.PLAIN]
Avoid using plain text passwords in Axis 'wsdd' files [CERT.MSC62.PTPT]
Ensure that passwords are not stored as plaintext and are sufficiently long [CERT.MSC62.PWDPROP]
Ensure that passwords are not stored as plaintext and are sufficiently long [CERT.MSC62.PWDXML]
Avoid using plain text passwords in Axis2 configuration files [CERT.MSC62.UTAX]
Avoid unencrypted passwords in WebSphere 'ibm-webservicesclient-ext.xmi' files [CERT.MSC62.WCPWD]
Avoid unencrypted passwords in WebSphere 'ibm-webservices-ext.xmi' files [CERT.MSC62.WPWD]
