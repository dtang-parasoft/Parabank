Avoid misspelling words in comments and property values [PROPS.SPELL]



DESCRIPTION 


This rule identifies misspelled words in comments and property values. 
An error is reported for each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v8.1
PARAMETERS 

-Ignore length of word less than (default value is 3)
-Ignore uppercase word (turned on by default)
-Ignore property file name (regular expression)
    -Allows you to provide a list of property files to ignore. If a property file 
     matches any of the regular expressions, it will be exempt from this rule.
BENEFITS 

Misspelled words in a comment will make the comment difficult to read and understand.
All misspelled word in property values should be corrected before the product release.
EXAMPLE 

#cuwmment	// VIOLATION
A=hwllo world	// VIOLATION
REPAIR 

#comment	// FIXED
A=hello world	// FIXED
