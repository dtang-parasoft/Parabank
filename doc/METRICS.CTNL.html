DEPRECATED: Number of lines in "class" or "interface" [METRICS.CTNL]



DESCRIPTION 


Rules from this category are deprecated - see METRIC category.

Counts the total number of lines in a class or interface, including comments,
blank lines, and "{ }" braces.
NOTES 

Jtest will report an error if the value is greater or equal 500.
BENEFITS 

Big classes are difficult to understand and maintain. Consider breaking down
big classes into smaller ones performing related tasks.
EXAMPLE 

We count from the line containing the class' opening "{" brace to the line
containing the class' closing "}" brace.

The following class counts as two lines:

package examples.rules.metrics;

public class CTNL {
}

//As does this one:

class CTNL2
{
}
