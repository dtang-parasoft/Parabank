Place a single space character or no space character between a method name and the opening "(" parenthesis [FORMAT.MSP]



DESCRIPTION 


This rule is a parameterized rule that identifies either:
1) method names that do not have a single space before their opening '(' parenthesis or 
2) method names that have a space before their opening '(' parenthesis. 
The rule parameters specify which of the above cases trigger the rule.
An error is reported for each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v4.5
NOTES 

This rule only applies to method declarations; it does not apply to method calls.
PARAMETERS 

-One space before the opening '(' parenthesis 
-No space before the opening '(' parenthesis (enabled by default)

The default "No space before the opening '('" option is in line with the default
pattern used by Eclipse and the older versions of the Parasoft tool.
BENEFITS 

If all team members follow the same formatting style, it will be easier for each
team member to read and understand the code.
EXAMPLE 

package examples.rules.format;

public class MSP {
    public void method() {  // VIOLATION
    }
    public void method1  () {  // VIOLATION
    }
}
REPAIR 

Place exactly one space between the method and its opening '(' parenthesis.

package examples.rules.format;

public class MSPFixed {
    public void method () {  // FIXED
    }
    public void method1 () {  // FIXED
    }
}
