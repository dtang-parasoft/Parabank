Avoid empty property values [PROPS.EMV]



DESCRIPTION 


This rule identifies any property value that is an empty string. An error
is reported for each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v8.1
PARAMETERS 

-Ignore property file name (regular expression)
    -Allows you to provide a list of property files to ignore. If a property file 
     matches any of the regular expressions, it will be exempt from this rule.
EXAMPLE 

a=  // VIOLATION
b=B
REPAIR 

a=A  // FIXED
b=B
