Place 'finalize()' methods between "public" and "protected" methods [CODSTA.ORG.ORFIM]



DESCRIPTION 


This rule identifies inappropriately-placed "finalize" methods.
An error is reported if a "finalize" method is not placed between
the "public" and "protected" methods of the class.
SCOPE LEVEL 
METHOD
SINCE 

v5.0
BENEFITS 

Makes code easier to read and less error-prone.
EXAMPLE 

package examples.rules.codsta.org;

public class ORFIM {
    public void method () {}
    protected void method2 () {}
    protected void finalize () {}  // VIOLATION
}
REPAIR 

Move the "finalize"  method.

package examples.rules.codsta.org;

public class ORFIMFixed {
    public void method () {}
    protected void finalize () {}  // FIXED
    protected void method2 () {}
}
