Use a naming convention for property names [PROPS.NAME]



DESCRIPTION 


This rule identifies property names that do not follow the
user-defined naming convention (as specified in a regular expression)
and property names that are too long. An error is reported for each occurrence.
SCOPE LEVEL 
LINE
SINCE 

v8.1
PARAMETERS 

-Max length of property name (default value is 80)
-regular expression
  (property names should match the listed regular expression.
   Default as "^[a-zA-Z0-9_\\.]+$")
-Negate regular expression (default is false)
-Ignore case (default is false)
-Ignore property file name (regular expression)
    -Allows you to provide a list of property files to ignore. If a property file 
     matches any of the regular expressions, it will be exempt from this rule.
BENEFITS 

This helps enforce a naming convention.
