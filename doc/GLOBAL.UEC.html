Avoid globally unused enum constants [GLOBAL.UEC]



DESCRIPTION 


This rule identifies unused enum constant.
An error is reported for each occurrence.

Global analysis is required in this rule.
SCOPE LEVEL 
LINE
SINCE 

v8.0
BENEFITS 

These unused entities usually point to either:

1) Old code that is no longer needed.  This makes the class more
   difficult to understand.

2) A logical flaw (if the entity needs to be used, but other classes
   incorrectly avoid using it).
DRAWBACKS 

The rule detects that enum constants are not explicitly used. Usage of values() or valueOf()
method is not treated as explicit usage and may lead in some rare cases to false positives.
EXAMPLE 

public enum UEC {
   UNUSED();  // VIOLATION
}
REPAIR 

Remove the unused field or document the reason for its existence.
