Do not use a package name that is reserved by Sun [NAMING.RPKG]



DESCRIPTION 


This rule identifies package names that are reserved by Sun. An 
error is reported for each occurrence.
SCOPE LEVEL 

FILE
SINCE 

v4.5
NOTES 

The standard Java distribution packages from Sun begin with the identifier 'java' 
or 'javax'. Sun has reserved this right so that the standard Java packages are
named in a consistent manner, regardless of the vendor of your Java development 
environment.
BENEFITS 

Following this convention prevents your API from being confused with Sun's API.
EXAMPLE 

package java.rules.naming;  // VIOLATION

public class RPKG {
}
REPAIR 

Use another package name.

package examples.rules.naming;  // FIXED

public class RPKGFixed {

}
