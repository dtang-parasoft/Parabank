Input-Based Attacks [SECURITY.IBA]

RULES

Do not extend from the Struts classes 'ActionForm' and 'DynaActionForm' [SECURITY.IBA.AEAF]
Avoid temporary files [SECURITY.IBA.ATF]
Avoid using "SELECT *" in SQL queries [SECURITY.IBA.AUSS]
Avoid parsing untrusted data with XMLDecoder [SECURITY.IBA.AUXD]
Canonicalize all data before validation [SECURITY.IBA.CDBV]
Always call 'super.validate()' from validation methods in 'ActionForm' classes [SECURITY.IBA.CSVFV]
Disable XML external entity injection [SECURITY.IBA.DXXE]
Encapsulate constructor arguments with a validation function [SECURITY.IBA.EDPM]
Use wrapper methods to secure native methods [SECURITY.IBA.NATIW]
Ensure proper session expiration [SECURITY.IBA.RUIM]
Disable LDAP deserialization [SECURITY.IBA.SC]
Use 'prepareCall' or 'prepareStatement' instead of 'createStatement' [SECURITY.IBA.UPS]
Encapsulate all redirect and forward URLs with a validation function [SECURITY.IBA.VRD]
Avoid XPath injection when evaluating XPath queries [SECURITY.IBA.XPIJ]
