Do not place empty whitespace at the end of a line [APSC_DV.003215.ATS]



DESCRIPTION 


This rule identifies code that has white space characters (space
or tab) at the end of the line. An error is reported for each 
occurrence.
SCOPE LEVEL 
LINE
SINCE 

v6.0
NOTES 

This rule ignores trailing spaces on comment lines.
PARAMETERS 

-Flag empty line containing extra whitespaces (default is false)
 If enabled, any empty line that contains whitespace characters (space or tab)
 would be flagged.
BENEFITS 

Trailing spaces are extra characters. Removing them can reduce file size.
REFERENCES 

DISA Application Security and Development Security Technical Implementation Guide (STIG)
APSC-DV-003215: The application development team must follow a set of coding standards
