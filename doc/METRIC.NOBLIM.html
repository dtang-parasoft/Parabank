Blank Lines in Methods [METRIC.NOBLIM]



DESCRIPTION 


Number of Blank Lines metrics count how many lines do not have code or comments.
A blank line is an empty line or line that has only whitespaces.
This metric counts number of blank lines that belong to methods.
It includes comments that precede the methods.

See METRIC.NOPLIM for details on how lines of Anonymous Methods are measured.

10.1.3
