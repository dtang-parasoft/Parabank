Ensure that JUnit 5 test classes that use @Ignore are annotated with @ExtendWith(IgnoreCondition.class) or @EnableJUnit4MigrationSupport [JUNIT.IGNORE]



DESCRIPTION 


"In order to provide a smooth migration path from JUnit 4 to JUnit Jupiter,
the junit-jupiter-migrationsupport module provides support for JUnit 4’s
@Ignore annotation analogous to Jupiter’s @Disabled annotation.
To use @Ignore with JUnit Jupiter based tests, configure a test dependency on
the junit-jupiter-migrationsupport module in your build and then annotate your
test class with @ExtendWith(IgnoreCondition.class) or
@EnableJUnit4MigrationSupport." [JUnit 5 User Guide]

This rule detects the @Ignore annotation in JUnit 5 test classes that are not
annotated with @ExtendWith(IgnoreCondition.class) or
@EnableJUnit4MigrationSupport.
SCOPE LEVEL 
LINE
SINCE 

v10.4.3
NOTES 

This rule assumes that a JUnit 5 test class is a class that contains a method
marked with one of the following annotations:
- @Test (from the org.junit.jupiter.api package)
- @TestFactory
- @TestTemplate,
- @ParameterizedTest
- @RepeatedTest
BENEFITS 

This rule facilitates the migration process from JUnit 4 to JUnit 5 by helping
you ensure that tests annotated with @Ignore will not be run after migration.
EXAMPLE 

In the following example, a JUnit 5 test class includes the @Ignore
annotation.


import org.junit.Ignore;
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

public class SomeTest
{
    @Test
    @Ignore // VIOLATION
    public void testBagMultiply()
    {
        Boolean a = Boolean.valueOf(true);
        Boolean b = Boolean.valueOf(false);

        assertTrue(a == b);
    }
}
REPAIR 

To fix the code, annotate the test class with
@EnableJUnit4MigrationSupport.

import org.junit.Ignore;
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;
import org.junit.jupiter.migrationsupport.EnableJUnit4MigrationSupport;

@EnableJUnit4MigrationSupport
public class SomeTest
{
    @Test
    @Ignore // NO VIOLATION
    public void testBagMultiply()
    {
        Boolean a = Boolean.valueOf(true);
        Boolean b = Boolean.valueOf(false);

        assertTrue(a == b);
    }
}
