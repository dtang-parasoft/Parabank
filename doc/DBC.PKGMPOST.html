Provide a '@post' contract for all package-private methods [DBC.PKGMPOST]



DESCRIPTION 


This rule identifies any package-private method without a '@post' contract. An 
error is reported for each occurrence.
SCOPE LEVEL 

METHOD
SINCE 

v4.5
NOTES 

If there's no Javadoc comment, Jtest will not report this error.
BENEFITS 

Post-conditions check whether the method works correctly.
EXAMPLE 

package examples.rules.dbc;

public class PKGMPOST
{
    /**
     * method add()
     */
    int add (int a, int b) {  // VIOLATION
        return a + b;
    }
}
REPAIR 

Provide a '@post' contract, or use '/** @post $none */' to 
explicitly document the lack of one.

package examples.rules.dbc;

public class PKGMPOSTFixed
{
    /** @post $result == a + b */  // FIXED
    int add (int a, int b) {
        return a + b;
    }
}
