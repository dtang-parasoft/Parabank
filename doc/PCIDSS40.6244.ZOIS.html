Inspect usage of scripting API [PCIDSS40.6244.ZOIS]



DESCRIPTION 


This rule identifies calls to ObjectInputStream.defaultReadObject.  It 
recommends that they be replaced with calls to ObjectInputStream.readFields.
SINCE 

v9.0
SECURITY RELEVANCE 

defaultReadObject can assign arbitrary objects to non-transient fields and does
not necessarily return.  readFields does not have these problems.
EXAMPLE 

package examples.rules.security.wsc;

import java.io.IOException;
import java.io.ObjectInputStream;

public class ZOIS {
    void method2(ObjectInputStream ois) throws IOException, ClassNotFoundException{
        ois.defaultReadObject();//VIOLATION
    }
}
REPAIR 

package examples.rules.security.wsc;

import java.io.IOException;
import java.io.ObjectInputStream;

public class ZOISFixed {
    void method2(ObjectInputStream ois) throws IOException, ClassNotFoundException{
        ois.readFields();//FIXED
    }
}
