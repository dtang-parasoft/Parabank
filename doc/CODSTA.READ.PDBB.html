Put declarations only at the beginning of blocks [CODSTA.READ.PDBB]



DESCRIPTION 


This rule enforces local variables to be declared only at the beginning
of blocks.
SCOPE LEVEL 
LINE
SINCE 

v5.0
NOTES 

One exception to this rule is "for" loop indices.
BENEFITS 

Not following this rule may confuse the unwary programmer and hamper code
portability within the scope.
EXAMPLE 

package examples.rules.codsta.read;

class PDBB {
    void foo () {
        String str = "hello world!";
        for (int i = 0; i < str.length(); i++) {
        }
        Object obj = new Object (); // VIOLATION
    }
}
REPAIR 

Declare local variables at the beginning of the block.

package examples.rules.codsta.read;

class PDBB_Fixed {
    void foo () {
        String str = "hello world!";
        Object obj = new Object (); 
        for (int i = 0; i < str.length(); i++) {
        }
    }
}
