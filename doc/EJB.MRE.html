Throw 'java.rmi.RemoteException' in the methods of remote interface and remote home interface [EJB.MRE]



DESCRIPTION 


This rule identifies methods of remote interface and remote home interface that do not throw 
'java.rmi.RemoteException'. An error is reported for each occurrence.
SCOPE LEVEL 

METHOD
SINCE 

v4.5
SECURITY RELEVANCE 

category: Erratic Application Behavior
category: Error Handling

Potential missed exception. See BENEFITS for more information.
BENEFITS 

Session and entity bean methods must follow the rules for RMI/IIOP. This means
that their argument and return values must be of valid types for RMI/IIOP,
and their thrown exceptions must include "java.rmi.RemoteException".
REFERENCES 

Sections 7.10.5 (p. 97) and 7.10.6 (p. 97) of the EJB specification (version 2.0). 
The EJB specification is available online at 
http://download.oracle.com/otndocs/jcp/7294-ejb-2.0-fr2-spec-oth-JSpec/
