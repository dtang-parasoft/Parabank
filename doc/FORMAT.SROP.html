Enforce number of space character(s) on each side of a relational operator [FORMAT.SROP]



DESCRIPTION 


This rule identifies relational operators that are not preceded and followed
by the required number of spaces. With the default parameters, an error
is reported for each relational operator that is not preceded and followed by
one space.
SCOPE LEVEL 
LINE
SINCE 

v4.5
PARAMETERS 

-Number of spaces before relational operators (default value is 1)
-Number of spaces after relational operators (default value is 1)
BENEFITS 

If all team members follow the same formatting style, it will be easier for each
team member to read and understand the code.
EXAMPLE 

package examples.rules.format;

public class SROP {
    public void method (int x, int y) {
        if (x<= y) { }  // VIOLATION
        if (x ==y) { }  // VIOLATION
    }
}
REPAIR 

Place a single space before and after the operator.

package examples.rules.format;

public class SROPFixed {
    public void method (int x, int y) {
        if (x <= y) { }  // FIXED
        if (x == y) { }  // FIXED
    }
}
