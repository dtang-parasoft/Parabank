Comment Lines in Files [METRIC.NOCLIF]



DESCRIPTION 


The Number of Comment Lines metrics measure how many lines have at least one
comment, have the beginning or ending of a comment, or are inside a comment.
The three metrics allow you to measure how well the analyzed code is commented.
This metric measures the number of comment lines in files.
SINCE 

10.1.3
EXAMPLE 

Metric  Code

+1      // comment import
        import java.io.Serializable;
        
+1      /**
+1       * comment Foo
+1       */
+1      public class Foo implements Serializable /* comment */
        {
            void bar() {
+1              // empty
            }
        }
        
Total: 6
