Throw 'javax.ejb.FinderException' in finder methods of remote home or local home interfaces [EJB.TFE]



DESCRIPTION 


This rule identifies any find<Method> method that is in a remote home or local
home interface but does not have a throws clause which includes 
"javax.ejb.FinderException". An error is reported for each occurrence.
SCOPE LEVEL 

METHOD
SINCE 

v4.5
REFERENCES 

Section 9.5.2 (p. 113) and 9.6.2 (p. 116) of the EJB specification (version 2.0).  
The EJB specification is available online at 
http://download.oracle.com/otndocs/jcp/7294-ejb-2.0-fr2-spec-oth-JSpec/
