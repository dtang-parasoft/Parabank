DESCRIPTION 


Ratio of Comments to Number of Logical Lines in types metrics report the ratio of comment lines
to logical lines of code.

See "Number of Comment Lines" metrics for the definition of comments. 
See "Number of Logical Lines" metrics for the definition of logical lines.

This metric measures the ratio of comment lines to logical lines of code in types.
