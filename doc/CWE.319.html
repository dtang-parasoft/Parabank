CWE-319: Cleartext Transmission of Sensitive Information [CWE.319]

RULES

Use 'https' instead of 'http' for the 'transportReceiver' and 'transportSender' in 'axis2.xml' configuration files [CWE.319.HTTPS]
Use the ''getSecure()'' and ''setSecure()'' methods to enforce the use of secure cookies [CWE.319.UOSC]
Use the SSL-enabled version of classes when possible [CWE.319.USC]
