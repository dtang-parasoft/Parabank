Blank Lines in Types [METRIC.NOBLIT]



DESCRIPTION 


Number of Blank Lines metrics count how many lines do not have code or comments.
A blank line is an empty line or line that has only whitespaces.
This metric counts number of blank lines that belong to types.
It includes comments that precedes the types.
SINCE 

10.1.3
