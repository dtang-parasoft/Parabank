APSC_DV.001740 The application must only store cryptographic representations of passwords. [APSC_DV.001740]

RULES

Avoid storing usernames and passwords in plain text in Castor 'jdo-conf.xml' files [APSC_DV.001740.PCCF]
Avoid using plain text passwords in Axis 'wsdd' files [APSC_DV.001740.PTPT]
Ensure that passwords are not stored as plaintext and are sufficiently long [APSC_DV.001740.PWDPROP]
Avoid using plain text passwords in Axis2 configuration files [APSC_DV.001740.UTAX]
Avoid unencrypted passwords in WebSphere 'ibm-webservicesclient-ext.xmi' files [APSC_DV.001740.WCPWD]
