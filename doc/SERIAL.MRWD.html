Ensure the return type of 'readResolve()' and 'writeReplace()' methods are 'java.lang.Object' [SERIAL.MRWD]



DESCRIPTION 


This rule indentifies the 'readResolve()' and 'writeReplace()' methods
declared in a 'java.io.Serializable' class which does not have the
correct return type 'java.lang.Object'. An error is reported for each
occurrence.
SINCE 

v8.2
BENEFITS 

Without the correct definition of the method, it will be ignored silently 
in the serialization mechanism
EXAMPLE 

package examples.rules.serial;

import java.io.*;

public class MRWD implements Serializable {

    protected Object[] readResolve() throws ObjectStreamException {  // VIOLATION
    	return null;
    }
}
REPAIR 

Change the return type to 'java.lang.Object'

package examples.rules.serial;

import java.io.*;

public class MRWDFixed implements Serializable {

    protected Object readResolve() throws ObjectStreamException { // FIXED
    	return null;
    }
}
