Avoid unnecessary calls to 'toString()' [CODSTA.READ.AUTS]



DESCRIPTION 


This rule identifies cases where a call to 'toString()' is unnecessary. The
following cases are checked:
    1) Object is concatenated to a String.
           For example:
           a) String s = "Error: " + errorObject.toString();
           b) String s = "Error: ";
              s += errorObject.toString();
    2) Object is passed to a "print" method of 'java.io.PrintStream' or
       'java.io.PrintWriter'.
           For example:
           a) System.err.println(errorObject.toString());
    3) Object is already a String.
           For example:
           a) "Hello world".toString();
SINCE 

v9.0
BENEFITS 

Removing the unnecessary 'toString()' call can simplify code and improve
readability.
EXAMPLE 

package examples.rules.codsta.read;

public class AUTS {
    void printError(Object errorObject) {
        System.err.println("Error: " + errorObject.toString()); //VIOLATION
    }
}
REPAIR 

Remove unnecessary 'toString()'.

package examples.rules.codsta.read;

public class AUTSFixed {
    void printError(Object errorObject) {
        System.err.println("Error: " + errorObject); //FIXED
    }
}
