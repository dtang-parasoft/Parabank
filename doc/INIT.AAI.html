Do not use array initializers [INIT.AAI]



DESCRIPTION 


This rule will flag the use of array initializers.
SCOPE LEVEL 
CLASS
SINCE 

v8.0
PARAMETERS 

-Allow empty array initializers (Default is true.)
BENEFITS 

Some people prefer initializing arrays element-by-element instead of using array
initializers.  Initializing arrays element-by-element can make the code easier
to read.
EXAMPLE 

package examples.rules.init;

public class AAI {
    public static void main(String[] args) {
        int[] a= {1, 2, 3};  //VIOLATION
    }
}
REPAIR 

Initialize the array element-by-element instead of using an array initializer.

package examples.rules.init;

public class AAIFixed {
    public static void main(String[] args) {
        int[] a= new int[3];  //FIXED
        a[0]= 1;
        a[1]= 2;
        a[2]= 3;
    }
}
