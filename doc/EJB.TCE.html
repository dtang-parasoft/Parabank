Throw 'javax.ejb.CreateException' in create methods of remote home or local home interfaces [EJB.TCE]



DESCRIPTION 


This rule identifies any create<Method> method that is in a remote home or
local home interface, but does not have a throws clause which includes
javax.ejb.CreateException. An error is reported for each occurrence.
SCOPE LEVEL 

METHOD
SINCE 

v4.5
REFERENCES 

Sections 7.10.6 (p. 98), 7.10.8 (p. 99), 9.5.1 (p. 112), and 9.6.1 (p. 115)
of the EJB specification (version 2.0).  The EJB specification is available 
online at http://download.oracle.com/otndocs/jcp/7294-ejb-2.0-fr2-spec-oth-JSpec/
