JavaBeans [BEAN] 

Rules related to JavaBean classes.
RULES

Use appropriate signatures for listener method names in JavaBean classes [BEAN.BLNC]
Override 'Object.equals()' in JavaBean classes [BEAN.EQUALS]
Do not use JDBC code in JavaBean classes [BEAN.JDBC]
Define get and set methods for each instance field [BEAN.NFM]
Ensure that JavaBean classes implement 'java.io.Serializable' [BEAN.SERIALIZABLE]
