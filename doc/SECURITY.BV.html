Backdoor Vulnerabilities [SECURITY.BV]

RULES

Do not access the class loader in a web component [SECURITY.BV.ACL]
Inspect usage of 'Date', 'Time' objects and 'System.currentTimeMillis()' method invocations [SECURITY.BV.ADT]
Inspect usage of 'getName()' from 'java.lang.Class' object [SECURITY.BV.AUG]
Use "read-only" AccessMode for Castor queries [SECURITY.BV.CQRO]
Do not set custom security managers outside of the 'main' method [SECURITY.BV.DSSM]
Ensure all sensitive method invocations are logged [SECURITY.BV.ENFL]
Do not call 'Socket.setSocketImplFactory()' or 'URL.setURLStreamHandlerFactory()' in a web component [SECURITY.BV.NSF]
Wrap "privileged" method invocations in "final" methods [SECURITY.BV.PCFM]
Wrap "privileged" method invocations in "private" methods [SECURITY.BV.PCPM]
Avoid using dynamically loaded classes in "privileged" code blocks [SECURITY.BV.PDLC]
Do not access or set System properties [SECURITY.BV.SYSP]
