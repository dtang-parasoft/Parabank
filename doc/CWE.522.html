CWE-522: Insufficiently Protected Credentials [CWE.522]

RULES

Avoid storing usernames and passwords in plain text in Castor 'jdo-conf.xml' files [CWE.522.PCCF]
Password information should not be included in properties file in plaintext [CWE.522.PLAIN]
Avoid using plain text passwords in Axis 'wsdd' files [CWE.522.PTPT]
Ensure that passwords are not stored as plaintext and are sufficiently long [CWE.522.PWDPROP]
Ensure that passwords are not stored as plaintext and are sufficiently long [CWE.522.PWDXML]
Protect against using unprotected credentials [CWE.522.TDPASSWD]
Avoid storing unencrypted Hibernate usernames and passwords in 'web.xml' files [CWE.522.UPWD]
Use the SSL-enabled version of classes when possible [CWE.522.USC]
Avoid using plain text passwords in Axis2 configuration files [CWE.522.UTAX]
Avoid unencrypted passwords in WebSphere 'ibm-webservicesclient-ext.xmi' files [CWE.522.WCPWD]
Avoid unencrypted passwords in WebSphere 'ibm-webservices-ext.xmi' files [CWE.522.WPWD]
