Portability [PORT] 

Rules related to Portability.
RULES

Avoid calling 'Thread.setPriority()' [PORT.CTSP]
Do not hard code an absolute pathname when calling a constructor from the 'File' class [PORT.DNHCP]
Do not use the non-portable 'System.getenv()' method [PORT.ENV]
Do not use "Runtime.exec()" [PORT.EXEC]
Do not hard-code IP addresses and port numbers [PORT.HCNA]
Do not hard code '\n' or '\r' as a line separator [PORT.LNSP]
Do not use user-defined "native" methods [PORT.NATV]
Do not use "java.awt.peer.*" interfaces directly [PORT.PEER]
Use "File.pathSeparator" or "File.pathSeparatorChar" instead of the corresponding literals [PORT.PSC]
