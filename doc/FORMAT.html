Formatting [FORMAT] 

Rules that enforce formatting style of code.
RULES

Use '()' to separate complex expressions [FORMAT.APAREN]
Ensure proper spacing in array references [FORMAT.ASPACE]
Do not place empty whitespace at the end of a line [FORMAT.ATS]
Put a blank line before each C-style comment [FORMAT.BLBC]
Enforce number of blank line(s) before type declarations [FORMAT.BLCD]
Enforce number of blank line(s) to separate "imports" from different packages [FORMAT.BLSIM]
Place a closing brace on its own line [FORMAT.CBRACE]
Place a single space character or no space character after type casting [FORMAT.CMS]
Do not leave spaces between qualified names and method invocations [FORMAT.DOT]
Use spaces instead of tabs (or tabs instead of spaces) [FORMAT.DUT]
Enforce the position of '{' brace [FORMAT.FCB]
Place the type that has the same name as the file as the first type [FORMAT.FCN]
Declare arrays with '[]' brackets after the array type and before the variable name(s) [FORMAT.IAD]
Enforce number of space(s) for indentation [FORMAT.IND]
Limit the maximum length of a line [FORMAT.LL]
Include a meaningful file header comment in every source file [FORMAT.MCH]
Enforce the order of annotations and modifiers [FORMAT.MO]
Place a single space character or no space character between a method name and the opening "(" parenthesis [FORMAT.MSP]
Do not place statements on the same line as the '{' opening brace [FORMAT.NSAB]
Write one statement per line [FORMAT.OSPL]
Enforce number of space character(s) after every comma [FORMAT.SAC]
Enforce number of space character(s) on each side of an assignment operator [FORMAT.SAOP]
Enforce number of space character(s) after the opening parenthesis "(" of a conditional statement [FORMAT.SAP]
Enforce number of space character(s) after every semicolon [FORMAT.SAS]
Enforce number of space character(s) between a prefixed unary operator and its operand [FORMAT.SAUOP]
Enforce number of space character(s) on each side of a bitwise operator [FORMAT.SBOP]
Enforce number of space character(s) between a postfixed unary operator and its operand [FORMAT.SBUOP]
Enforce number of space character(s) between a conditional statement and the opening "(" parenthesis [FORMAT.SC]
Enforce number of space character(s) before and after the "?" conditional operator [FORMAT.SCOP]
Enforce number of space character(s) on each side of a logical operator [FORMAT.SLOP]
Enforce number of space character(s) on each side of a relational operator [FORMAT.SROP]
Avoid using trailing comments [FORMAT.TC]
Avoid or enforce the use of trailing commas in array initializers [FORMAT.TCOMMA]
Use the preferred formatting for conditional expressions [FORMAT.TE]
Make sure all files are terminated with a newline character [FORMAT.TNL]
Enforce number of blank line(s) between major sections [FORMAT.U2BL]
Avoid unnecessary parentheses [FORMAT.UP]
