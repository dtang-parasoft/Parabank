Do not extend other @MessageDriven beans from a @MessageDriven bean [EJB.EJB3.MDBS]



DESCRIPTION 


This rule identifies @MessageDriven beans which have another @MessageDriven
bean as a superclass. The EJB3 Specification does not provide this capability.
SINCE 

v8.4
PARAMETERS 

Check the following:
    -Check annotations
    -Check annotations and deployment descriptors (default)
        Check deployment descriptors in the following location:
            -Check project (default)
            -Check package
EXAMPLE 

package examples.rules.ejb.ejb3;

@MessageDriven
public class MDBS extends MessageDrivenClass { //VIOLATION
    ...
}

@MessageDriven
class MessageDrivenClass {
    ...
}
REPAIR 

Remove extension of @MessageDriven superclass.

package examples.rules.ejb.ejb3;

@MessageDriven
public class MDBSFixed { //FIXED
    ...
}
REFERENCES 

Debu Panda, Reza Rahman, and Derek Lane. EJB3 in Action.
