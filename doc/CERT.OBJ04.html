OBJ04-J: Provide mutable classes with copy functionality to safely allow passing instances to untrusted code [CERT.OBJ04]

RULES

Make your 'clone()' method "final" for security [CERT.OBJ04.CLONE]
Enforce returning a defensive copy in 'clone()' methods [CERT.OBJ04.CPCL]
Do not pass user-given mutable objects directly to certain types [CERT.OBJ04.MPT]
Provide mutable classes with copy functionality [CERT.OBJ04.MUCOP]
Do not store user-given mutable objects directly into variables [CERT.OBJ04.SMO]
