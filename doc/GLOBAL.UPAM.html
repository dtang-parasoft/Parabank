Avoid globally unused package-private methods [GLOBAL.UPAM]



DESCRIPTION 


This rule identifies unused package-private methods. An error is reported for
each occurrence.

Global analysis is required in this rule.
SCOPE LEVEL 
METHOD
SINCE 

v4.5
SECURITY RELEVANCE 

category: Weak Security Controls
category: Code Quality

The unused code might implement security features. Security enhancement will be
missed if the code is not used.
BENEFITS 

These unused entities usually point to either:

1) Old code that is no longer needed.  This makes the class more
   difficult to understand.

2) A logical flaw (if the entity needs to be used, but other classes
   incorrectly avoid using it).
REPAIR 

Remove the unused method or document the reason for its existence.
